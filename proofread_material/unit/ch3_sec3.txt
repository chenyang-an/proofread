\section{Structures and the Definition of Truth}\label{sec:truthFO}

\subsection{Structures}

The definition of truth is considerably more complicated for first-order logic
than for propositional logic. In propositional logic, formulas
contain the logical symbols $\lnot, \lor, \land, \limplies, \liff$
that have fixed meanings and variables~$p_i$ that range over only two possible
values. To define a truth value for a propositional formula, all that
was necessary was to start with a truth assignment~$\varphi$
that gives true/false values to the variables. In contrast, first-order formulas use
the logical symbols $\lnot, \lor, \land, \limplies, \liff, \forall, \exists$
and the equality symbol~$=$ that have fixed meanings. They also use
variables~$x_i$ that range over some domain of objects as well as
constant, function, and predicate symbols. Before a first-order
formula can be given a truth value, it is necessary to specify a domain
of objects, to specify which objects are denoted by the
constant symbols, and to specify the values of all the functions and
predicates for all inputs from the domain. All this information
is called an ``\idxnm{interpretation}'' or a ``\idxnm{structure}''.
If there are free variables in the formula, it is also
necessary to specify which objects are denoted by the free variables.
This information is called an ``\idxnm{object assignment}''.

The original definition of truth for first-order formulas in a general setting was
given by A.\ Tarski in 1933. Perhaps surprisingly, this was well after
first-order languages and proofs were well-developed. The definition of
truth given below
is quite close to Tarski's original conception. It proceeds in three stages.
First, it is necessary to define the values of terms as objects in
the domain of individuals. This uses the interpretations of
functions and the values assigned to variables and constant symbols.
Second, it is necessary to define the truth
of atomic formulas. This uses the interpretations of predicate symbols.
Finally, a recursive definition defines the
truth of an arbitrary formula by using the usual meanings of the
logical connectives $\lnot, \lor, \land, \limplies, \liff, \forall, \exists$.

But first, we must define the notion of ``\idxnm{interpretation}'' or ``\idxnm{structure}''.
(The two terms are synonymous.) Interpretations or structures are also sometimes
called ``models''\index{model}.
It is common to use a variety of different symbols to denote
structures, including $\mathfrakA,\,\mathfrakB,\,\mathfrakC$ (the Fraktur letters ``A'', ``B'', and~``C'') or
$\mathcal M,\,\mathcal N$ (``M'' stands for ``model'') or
$\mathcal I,\,\mathcal J$ (``I''~stands for ``interpretation'').
We work with a fixed language~$L$.

\begin{definition}
Fix a first-order language~$L$. An \emph{$L$-\idxnm{structure}}~$\mathfrakA$ consists of the following items:
\begin{enumerateLoose}
\item A nonempty set called the \idxem{domain} or \idxem{universe}
of~$\mathfrakA$. This set
is denoted~$|\mathfrakA|$.
This is the set of {\em objects} that variables can range over.
\item For each \idxnm{constant symbol}~$c$ in~$L$,
a member of the universe~$|\mathfrakA|$. This member is
denoted $c^\mathfrakA$ and is called the \idxem{interpretation} of~$c$ in~$\mathfrakA$.
\item For each $k$-ary \idxnm{predicate symbol}~$P$ in~$L$, a
subset of $|\mathfrakA|^k$; that is, a set of $k$-tuples
of members of~$|\mathfrakA|$. This subset is
a ``$k$-ary \idxnm{relation}'' on~$|\mathfrak A|$;
it is denoted~$P^\mathfrakA$ and is called the \idxem{interpretation} of~$P$ in~$\mathfrakA$.
\item For each $k$-ary \idxnm{function symbol}~$f$ in~$L$,
a subset of~$|\mathfrakA|^{k+1}$ that is the
\idxnm{graph of a function}\index{function}~$f: |\mathfrakA|^k \rightarrow |\mathfrakA|$.
This subset is denoted~$f^\mathfrakA$ and is called the
\idxem{interpretation} of~$f$ in~$\mathfrakA$.

What it means for $f^\mathfrakA$ to be ``the \idxnm{graph of a function}''
is that $f^\mathfrakA$ is a set of $(k+1)$-tuples,
and that for every sequence $a_1,\ldots, a_k$
of $k$~members of~$|\mathfrakA|$, there is
a unique~$a_{k+1}$ such that
$\langle a_1,\ldots, a_k,a_{k+1} \rangle \in f^\mathfrakA$.

The notation $f^\mathfrakA$ is used to denote both the \idxnm{graph of a function}
and the \idxnm{function} itself.  In particular, we write
\[
f^\mathfrakA(a_1, \ldots, a_k )
\]
to denote the unique value~$a_{k+1}$ such that
$\langle a_1,\ldots,a_k,a_{k+1} \rangle \in f^\mathfrakA$.
\end{enumerateLoose}
\end{definition}

The $L$-structure thus consists of the specifications
of~$|\mathfrakA|$ and of all the values of $c^\mathfrakA$,
$P^\mathfrak|$, and $f^\mathfrakA$.
The intuition is that the constant symbol~$c$
denotes the object~$c^\mathfrakA$ in~$|\mathfrakA|$.
Likewise, the intuition is that $P^\mathfrakA$ is the set of
tuples for which $P(a_1,\ldots, a_k)$ is true in~$\mathfrakA$.
Or, informally,\footnote{The notation $\langle a_1,\ldots, a_k\rangle$
denotes a $k$-tuple of objects. When $k=1$, the convention is
that $\langle a_1 \rangle$ is the same object as $a_1$ itself, so
we can use ``$\langle a_1 \rangle$'' and ``$a_1$'' interchangeably.}
\[
\hbox{$P(a_1,\ldots, a_k)$ is true for $\mathfrakA$
~~~if and only if~~~ $\langle a_1,\ldots, a_k\rangle \in P^\mathfrakA$.}
\]
Finally, the intuition is that
\[
a_{k+1} = f(a_1,\ldots,a_k)~\hbox{is true for~$\mathfrakA$} \quad \hbox{if and only if}\quad
\langle a_1,\ldots, a_k,a_{k+1} \rangle \in f^\mathfrakA ,
\]
namely, that $f(a_1,\ldots,a_k)$ is equal to
$f^\mathfrakA(a_1,\ldots,a_k)$.
These informal intuitions will be made formal below in the
definition of truth.

\begin{example}\label{example:structureN}
Consider the language of arithmetic, $L_\PA$, with
non-logical symbols $0, S, +, \cdot$, as defined in
Example~\ref{example:languagePA}.  We give two possible
$L_\PA$-structures.
The first one, denoted~$\mathcal N$ corresponds to the
nonnegative integers. The universe~$| \mathcal N |$ of~$\mathcal N$
is the set of nonnegative integers, $S$~is interpreted as
the ``increment by~1'' function, and the symbols
$0$, $+$ and~$\cdot$ have their usual interpretations.
Formally, we let
\begin{eqnarray*}
|\mathcal N| &=& \hbox{$\mathbb N$, i.e., the set of nonnegative integers.} \\
0^{\mathcal N} &=& \hbox{$0$, i.e., the integer~0.} \\
S^{\mathcal N} &=& \{ \langle i,j \rangle: i,j\in \mathbb N,\, j=i+1 \} \\
+^{\mathcal N} &=& \{ \langle i,j,k \rangle: i,j,k\in \mathbb N,\, i+j=k \} \\
\cdot^{\mathcal N} &=& \{ \langle i,j,k \rangle: i,j,k\in \mathbb N,\, i\cdot j = k \}
\end{eqnarray*}
\end{example}

\begin{example}\label{example:structureZ}
The second $L_\PA$-structure is the set~$\mathbb Z$ of
all integers with the usual meanings for $0$, $+$, and~$\cdot$.
We denote this structure~$\mathcal Z$.
Formally, we let
\begin{eqnarray*}
|\mathcal Z| &=& \hbox{$\mathbb Z$, i.e., the set of all integers.} \\
0^{\mathcal Z} &=& \hbox{$0$, i.e., the integer~0.} \\
S^{\mathcal Z} &=& \{ \langle i,j \rangle: i,j\in \mathbb Z,\, j=i+1 \} \\
+^{\mathcal Z} &=& \{ \langle i,j,k \rangle: i,j,k\in \mathbb Z,\, i+j=k \} \\
\cdot^{\mathcal Z} &=& \{ \langle i,j,k \rangle: i,j,k\in \mathbb Z,\, i\cdot j = k \}
\end{eqnarray*}
\end{example}
The structures $\mathcal N$ and $\mathcal Z$ are not isomorphic, and the truth of sentences
and formulas
can be different in $\mathcal N$ and~$\mathcal Z$. For instance, the two sentences
\[
\forall x\, \exists y\, ( S(y) = x )
\qquad \hbox{and} \qquad
\forall x \,\exists y \, (x+y=0 \land y+x=0)
\]
are true in~$\mathcal Z$ but false in~$\mathcal N$.
The first sentence states that
every object~$x$ is the successor of some~$y$; the second
states that every object has an additive inverse.
Conversely, the negations of these two sentences are
true in~$\mathcal N$ and false in~$\mathcal Z$.

\begin{example}\label{example:structureZ3} \index{group theory|(}
Let $L$ be the language of groups with
nonlogical symbols $1$, $\cdot$ and~${}^{-1}$. The
integers with addition mod~3 can be viewed as an
$L$-structure~$\mathcal Z_3$.  For this, 
the universe is $|{\mathcal Z_3}| = \{0,1,2\}$
and the group operations
are given by:
\[
\hbox{
\begin{tabular}{c|ccc}
$\cdot^{\mathcal Z_3}$ & 0 & 1 & 2 \\
\hline
0 & 0 & 1 & 2 \\
1 & 1 & 2 & 0 \\
2 & 2 & 0 & 1
\end{tabular}
\hspace*{2em} and \hspace*{2em}
\begin{tabular}{c|ccc}
 & $({}^{-1})^{\mathcal Z_3}$   \\
 \hline
0 & 0  \\
1 & 2  \\
2 & 1
\end{tabular}
}
\]
The corresponding structure~$\mathcal Z_3$ is formally specified as
\begin{eqnarray*}
1^{\mathcal Z_3} &=& 0 \hbox{\quad (the identity element for the group operation)} \\
\cdot^{\mathcal Z_3} &=& \{ \langle 0,0,0 \rangle, \langle 0,1,1 \rangle, \langle 0,2,2 \rangle,
                            \langle 1,0,1 \rangle, \langle 1,1,2 \rangle, \langle 1,2,0 \rangle, \\
&& \quad \langle 2,0,2 \rangle, \langle 2,1,0 \rangle, \langle 2,2,1 \rangle  \} \\
({}^{-1})^{\mathcal Z_3} &=& \{ \langle 0,0 \rangle, \langle 1,2 \rangle, \langle 2,1 \rangle \}
\end{eqnarray*}
\end{example}
The last example is potentially confusing since the language~$L$ uses multiplicative
notation, but the structure is based on the group of order three which is usually
described as an {\em additive} group, namely addition mod~3. However, the point is that
the symbols of a language can stand for anything.  That is, a $k$-ary function
symbol can be interpreted in a structure as {\em any} $k$-ary function on the
universe; a $k$-ary predicate symbol can be interpreted as {\em any} $k$-ary
predicate on the universe (a set of $k$-tuples), and a constant symbol
can be interpreted as {\em any} member of the universe.  In the last example,
the symbol~$\cdot$ could be interpreted as any binary operation
on the universe, and then first-order statements can express simple
properties such as whether $\cdot$ is associative or commutative, etc.
This is similar in spirit
to the abstract approach taken in group theory, or more generally in
algebra.

The example chose the interpretation~$\cdot ^ {\mathcal Z_3}$ of
the binary function symbol~$\cdot$ to be the
usual addition operation on~$\mathcal Z_3$, and the interpretation~$1^{\mathcal Z_3}$
of the constant symbol~$1$ to be the identity element~$0$ of~$\mathcal Z_3$. This is
also potentially confusing since~``$1$'' is being used in two ways: First, ``1''~is a
{\em symbol} in the language~$L$. Second, ``1''~is an {\em element} (in the
universe) of~$\mathcal Z_3$.
These are completely different meanings for~``1'', and the reader should be careful
to distinguish between them.
\index{group theory|)}

\subsection{Definition of truth}\label{sec:defnTruthFO}
\index{truth, definition of|(}

\paragraph*{Object assignments.}
Our next goal is to define the truth of an $L$-formula or an $L$-sentence
in a structure. Suppose, for example, that we wish to determine
whether the $L$-sentence
\begin{equation}\label{eq:objectAssignEx}
\forall x\, \exists y\, (x\cdot y = x)
\end{equation}
is true in a given $L$-structure. Our definition of truth
will use induction on the complexity of the formula~(\ref{eq:objectAssignEx}).
For this reason, we must define the truth of its subformula
$\exists y( x\cdot y = x )$. But this is a {\em formula}, not a sentence,
since it uses the free variable~$x$. In particular,
the truth of $\exists y( x\cdot y = x)$ depends on which object
in the universe $|\mathfrakA|$ is denoted by~$x$.  This will be specified by
a function called an ``object assignment'' that maps variables
to objects in the universe:
\begin{definition}\label{def:obhjectAssign}
Let $\mathfrakA$ be a structure. An \idxem{object assignment} for~$\mathfrakA$
is a function~$\sigma$ with domain the set of variables $x_1, x_2, x_3,\ldots$
and with range the universe of~$\mathfrakA$; namely,
\[
\sigma : \{x_1,x_2,x_3,\ldots \} \rightarrow |\mathfrakA|.
\]
\end{definition}
The point of an object assignment~$\sigma$ is that $\sigma(x_i)$
specifies which member of the universe
is denoted by~$x_i$.

\paragraph*{Denotation of terms.}
The first step in the definition of truth is to define what objects
are denoted by terms.  We start with an object assignment~$\sigma$
and extend it to have domain the set of $L$-terms, so that each $L$-term
is assigned a value from the universe~$|\mathfrakA|$. The idea is
to use the interpretations of the constant symbols and function symbols
as given by the structure~$\mathfrakA$, and use these to compute
the value of a term in the most straightforward way possible.

\begin{definition}\index{term}
Suppose $\sigma$ is an \idxnm{object assignment} for the $L$-structure~$\mathfrakA$.
The function~$\sigma$ is extended to have domain the set
of all $L$-terms~$t$ by defining $\sigma(t)$ by recursion as:
\begin{enumerateLoose}
\item If $t$ is a constant symbol $c\in L$, then
$\sigma(t) = c^\mathfrakA$.
\item If $t$ is of the form $f(t_1, t_2, \ldots, t_k)$ for a $k$-ary function symbol $f\in L$, then
\[
\sigma(t) ~=~ f^\mathfrakA(\sigma(t_1), \sigma(t_2), \ldots, \sigma(t_k)) .
\]
\end{enumerateLoose}
\end{definition}

\begin{example}
Let $\mathcal Z_3$ be the structure described in Example~\ref{example:structureZ3}.
Consider the term $(x\cdot y)^{-1}\cdot x$, and suppose $\sigma(x) = 1$
and $\sigma(y) =2$.  Then $\sigma(x\cdot y) = 0$, and $\sigma((x\cdot y)^{-1}) = 0$,
and $\sigma((x\cdot y)^{-1}\cdot x) = 1$.
\end{example}

The value of $\sigma(t)$ depends on
the structure~$\mathfrakA$ in addition to $\sigma$ and~$t$;
in particular, $\sigma(t)$ depends on the
interpretations~$f^\mathfrakA$ and the function symbols~$f$ that appear in~$t$.
However, we will generally be working with a particular~$\mathfrakA$, so
there will usually be no confusion in suppressing $\mathfrakA$ in the
notation and writing just ``$\sigma(t)$''.

Note that the value of $\sigma(t)$ depends only
on the values of $\sigma(x_i)$ for the variables~$x_i$ that
actually appear in~$t$. This should be obvious of course and can be proved
by induction on the complexity of terms. If $t$ is a closed term,
then $\sigma(t)$ is independent of the choice of~$\sigma$,
it only depends on~$\mathfrakA$. In that case, we sometimes write
$t^\mathfrakA$ instead of~$\sigma(t)$:

\begin{definition}\label{def:tClosedValue}
If $t$ is a closed term, then $t^\mathfrakA$ denotes $\sigma(t)$ where
$\sigma$ is an arbitrary object assignment.
\end{definition}

\paragraph*{Definition of truth for atomic formulas.}\index{atomic formula}
The definition of truth for general $L$-formulas starts
with the definition of truth for atomic formulas.
We use the notation $\mathfrakA \vDash A[\sigma]$ to indicate
that $A$ is true in the structure with the object assignment~$\sigma$.
\begin{definition}\index{atomic formula}
Let $\mathfrakA$ be an $L$-structure, $\sigma$~be an object assignment,
and $A$~be an atomic $L$-formula. We define \emph{$A$~is true
in~$\mathfrakA$ with object assignment~$\sigma$}, written
$\mathfrakA \vDash A[\sigma]$\index{double turnstile@{double turnstile ($\vDash$)}}, 
as follows:
\begin{enumerateLoose}
\item If $A$ is a formula $s=t$, where $s$ and~$t$ are $L$-terms then,
\[
\mathfrakA \vDash A[\sigma] \quad\hbox{if and only if}\quad \sigma(s) = \sigma(t).
\]
\item If $A$ is a formula $P(t_1, \ldots t_k)$ where $P$~is a $k$-ary predicate
and the $t_i$'s are $L$-terms, then
\[
\mathfrakA \vDash A[\sigma] \quad\hbox{if and only if}\quad \langle \sigma(t_1), \ldots, \sigma(t_k) \rangle \in P^\mathfrakA.
\]
\end{enumerateLoose}
\end{definition}

\begin{example}\label{example:atomicTruth}
Let $L$ the language of directed graphs that includes the nonlogical binary
predicate symbol $E(\cdot, \cdot)$, so $E(x,y)$ indicates there is
a directed edge from vertex~$x$ to vertex~$y$.  Figure~\ref{fig:exampleAtomicTrue}
shows an $L$-structure~$\mathfrakA$ with $E^\mathfrakA$ equal to the 
set $\{ \langle a, b \rangle, \allowbreak 
\, \langle a, c \rangle, \allowbreak \, 
\langle b, c \rangle, \allowbreak \, 
\langle c, b \rangle \} $.
Consider the atomic $L$-formula $E(x_1, x_2)$. This formula is true in~$\mathfrakA$
under an object assignment~$\sigma$ precisely when one of the following
four conditions holds: 
(a)~$\sigma(x_1)=a$ and $\sigma(x_2) = b$,
(b)~$\sigma(x_1)=a$ and $\sigma(x_2) = c$,
(c)~$\sigma(x_1)=b$ and $\sigma(x_2) = c$, or
(d)~$\sigma(x_1)=c$ and $\sigma(x_2) = b$.

That is, $\mathfrakA \vDash E(x_1,x_2) [\sigma]$ precisely when one of these
four conditions holds.
\end{example}

\paragraph*{Definition of truth for general first-order formulas.}
We are almost ready to give the definition of truth for
general formulas~$A$.  We will continue to write
$\mathfrakA \vDash A[\sigma]$ to denote that $A$~is true in~$\mathfrakA$
with object assignment~$\sigma$.
First, however, we need to define the notion
of an ``$x_i$-variant\index{variant!xivariant@{$x_i$-variant}}'' of an object assignment.

\begin{definition}\label{def:xivariant}
Let $\sigma$ be an object assignment. An \emph{$x_i$-variant\index{variant!xivariant@{$x_i$-variant}}} of~$\sigma$
is an object assignment~$\tau$ such that $\tau(x_j) = \sigma(x_j)$ for
all $j\neq i$.
\end{definition}
An $x_i$-variant is allowed to have $\tau(x_i)$ be any member of
the universe~$|\mathfrakA|$.
Note that it is permitted that $\tau(x_i)$ is equal to~$\sigma(x_i)$; in this case,
$\tau$ and~$\sigma$ are identical.

We use $x_i$-variants to handle
the definition of truth for quantifiers.
To motivate this, let $A$ be the formula $\forall x_i\, (x_j\le x_i)$ and
suppose we wish to define $\mathfrakA \vDash A[\sigma]$, namely
the truth of $A$
in a structure~$\mathfrakA$ with object assignment~$\sigma$.
The variable~$x_j$ is free in~$A$, so the value of $\sigma(x_j)$ is
needed when defining $\mathfrakA \vDash A[\sigma]$. However, $x_i$~is
not free in~$\sigma$, and thus $\sigma(x_i)$ will have no bearing on
whether $\mathfrakA \vDash A[\sigma]$ holds.
The recursive definition of truth will define $\mathfrakA \vDash A[\sigma]$
in terms of $\mathfrakA \vDash A [\tau]$ where $\tau$~ranges
over all $x_i$-variants of~$\sigma$. In other words, the value $\tau(x_j)$ is kept
fixed and equals $\sigma(x_j)$, but all possible values for $\tau(x_i)$
must be considered. The point is that since $A$~starts with the quantifier $\forall x_i$,
it is necessary to consider all possible values for $\tau(x_i)$ to evaluate
the truth of~$A$.


The condition $\mathfrakA \vDash A[\sigma]$ is defined inductively, using
recursion on the complexity of~$A$.
The case base is when $A$~is atomic, and the inductive steps handle
the propositional connectives $\lnot\penalty10000 , \land\penalty10000 ,\penalty1000 \lor\penalty10000,\penalty1000 \limplies\penalty10000,\liff$
and the quantifiers $\forall, \exists$.

We write
$\mathfrakA \nvDash A[\sigma]$\index{double turnstile@{double turnstile ($\vDash$)}!negated ($\nvDash$)}
to denote
that $\mathfrakA \vDash A[\sigma]$ is false.

\begin{definition}\label{def:truthFO}
Let $\mathfrakA$ be an $L$-structure, $\sigma$~be an object assignment,
and $A$~be an atomic $L$-formula. If $A$ is atomic,
$\mathfrakA \vDash A[\sigma]$ has already been defined.
For non-atomic~$A$, we recursively define \emph{$A$~is true
in~$\mathfrakA$
with object assignment~$\sigma$}, written
$\mathfrakA \vDash A[\sigma]$\index{double turnstile@{double turnstile ($\vDash$)}},
as follows:

\def\tempLayout#1#2{
\parbox{1in}{\hfill $#1$}\qquad \hbox{if and only if} \qquad\parbox{3in}{$#2$.\hfill}
}

\begin{enumerateLooser}
\item If $A$ is $\lnot B$, then $\mathfrakA \vDash A[\sigma]$ holds if and only if
\[
\mathfrakA \nvDash B[\sigma].
\]
\item If $A$ is $(B \lor C)$, then $\mathfrakA \vDash A[\sigma]$ holds if and only if
\[
\mathfrakA \vDash B[\sigma] \hbox{~or~} \mathfrakA \vDash C[\sigma].
\]
\item If $A$ is $(B \land C)$, then $\mathfrakA \vDash A[\sigma]$ holds if and only if
\[
\mathfrakA \vDash B[\sigma] \hbox{~and~} \mathfrakA \vDash C[\sigma].
\]
\item If $A$ is $(B \limplies C)$, then $\mathfrakA \vDash A[\sigma]$ holds if and only if
\[
\mathfrakA \nvDash B[\sigma] \hbox{~or~} \mathfrakA \vDash C[\sigma]
\]
\item If $A$ is $(B \liff C)$, then $\mathfrakA \vDash A[\sigma]$ holds if and only if
\[
\hbox{$\mathfrakA \vDash B[\sigma]$ and $\mathfrakA \vDash C[\sigma]$
           are both true or both false.}
\]
\item If $A$ is $\exists x_i\, B$, then $\mathfrakA \vDash A[\sigma]$ holds if and only if
\[
\hbox{for some $x_i$-variant~{$\tau$} of~$\sigma$, $\mathfrakA \vDash B[\tau]$.}
\]
\item If $A$ is $\forall x_i\, B$, then $\mathfrakA \vDash A[\sigma]$ holds if and only if
\[
\hbox{for every $x_i$-variant~{$\tau$} of~$\sigma$, $\mathfrakA \vDash B[\tau]$.}
\]
\end{enumerateLooser}
\end{definition}

The definition of truth for
the propositional connectives in (a)-(e) is identical to the way
Definition~\ref{def:truthProp} defined the truth of propositional
variables.  Cases (f) and~(g) handle the truth of quantified formulas
by using $x_i$-variants~$\tau$ of~$\sigma$. As already discussed, these $x_i$-variants
change the object assignment~$\sigma$ to map $x_i$ to an arbitrary
member of~$|\mathfrakA|$.

\begin{example}\label{example:fmlaTruthGraph}
Returning to the $L$-structure~$\mathfrakA$ discussed in Example~\ref{example:atomicTruth}
and shown in Figure~\ref{fig:exampleAtomicTrue}, consider the formula
\[
\exists x_1\, \forall x_2\, (x_1\not=x_2 \limplies E(x_1,x_2)).
\]
From the definition of truth for~$\limplies$,
we see that $\mathfrakA\vDash (x_1\not=x_2 \limplies E(x_1,x_2))[\sigma]$
holds if and only if either $\sigma(x_1)=\sigma(x_2)$ or if one
of the conditions (a)-(d) of Example~\ref{example:atomicTruth} holds.

From this, it follows that $\mathfrakA\vDash \forall x_2\, (x_1\not=x_2 \limplies E(x_1,x_2))[\sigma]$
holds precisely if $\sigma(x_1) = a$.  Finally, we
have that
$\mathfrakA\vDash \exists x_1\, \forall x_2\, (x_1\not=x_2 \limplies E(x_1,x_2))[\sigma]$
holds for every~$\sigma$, since every $\sigma$ has an $x_1$-variant~$\sigma'$ 
such that $\sigma'(x_1) = a$.
\end{example}

The truth of $\mathfrakA \vDash A[\sigma]$ depends only the
values of $\sigma(x_i)$ for those variables~$x_i$ that appear free in~$A$;
it does not depend on the values $\sigma(x_i)$ for variables~$x_i$ that
do not appear free in~$A$. Informally, this is because the $x_i$-variants~$\tau$
that handle quantifiers $\forall x_i$ and~$\exists x_i$ ``overwrite''
the value of~$\sigma(x_i)$. It also reflects our informal understanding
that  bound variables could be renamed arbitrarily and not affect the
truth of $\mathfrakA \vDash A[\sigma]$; that is, bound variables can be
renamed arbitrarily as long as there is no clash between variables.
For a precise statement of this, see Theorem~\ref{thm:renameBoundVars} later in
this chapter.

The next theorem formalizes the idea that the truth of
$\mathfrakA \vDash A[\sigma]$ depends only the
values of $\sigma(x_i)$ for $x_i$'s that appear free in~$A$.
\begin{theorem}\label{thm:dependsOnFree}
Let $\sigma$ and~$\tau$ be object assignments such
that $\sigma(x_i) = \tau(x_i)$ for every variable~$x_i$
that appears free in~$A$. Then
$\mathfrakA \vDash A[\sigma]$ if and only if
$\mathfrakA \vDash A[\tau]$.
\end{theorem}

The proof of Theorem~\ref{thm:dependsOnFree} is left to the
reader. The first stage of the proof uses induction on terms to prove that,
for all~$\sigma$, the value of~$\sigma(t)$
depends only on the values~$\sigma(x_i)$ for
variables~$x_i$ that actually appear in~$t$. The theorem
is then proved by induction on the complexity of~$A$.

Recall that a sentence is a formula with no free occurrences of variables.
Theorem~\ref{thm:dependsOnFree} immediately gives:
\begin{corollary}
If $A$ is a sentence, then the truth of $\mathfrakA \vDash A[\sigma]$
does not depend on the choice of~$\sigma$.
\end{corollary}

The corollary justifies the next definition; namely, we can omit using
an object assignment~$\sigma$ when discussing the truth of a
{\em sentence}.

\begin{definition}
Let $A$ be an $L$-sentence and $\mathfrakA$ an $L$-structure.
The \emph{$A$ is true in~$\mathfrakA$}\index{truth, definition of},
written $\mathfrakA \vDash A$, if and only if
$A$~is true in~$\mathfrakA$ under any object assignment~$\sigma$.
\end{definition}

\begin{example}\label{example:Z3true}
This example and the next example work with the group $\mathcal Z_3$
of Example~\ref{example:structureZ3}.  First, we consider the
following sentence~$A$:
\begin{equation}\label{eq:exampleZ3true}
\exists x_1\, \forall x_2 \, (x_1\cdot x_2 = x_2).
\end{equation}
Note that this asserts the existence of a left identity.
We claim that $\mathcal Z_3 \vDash A [\sigma]$ for every object assignment~$\sigma$.
Since $A$~is a sentence, we can also write this as $\mathcal Z_3 \vDash A$, without needing
to mention the object assignment~$\sigma$.
To prove this, we must show that
\begin{equation}\label{eq:exampleZ3true2}
\mathcal Z_3 \vDash \forall x_2 \, (x_1\cdot x_2 = x_2) [\tau ]
\end{equation}
is true for some $x_1$-variant~$\tau$ of~$\sigma$.  There are three $x_1$-variants~$\tau$ that
might be considered, as we can have $\tau(x_1)=0$ or $\tau(x_1)=1$ or $\tau(x_1)=2$.
Since are trying to show that (\ref{eq:exampleZ3true}) is true, and since $x_1$ is
{\em existentially} quantified, it is enough to work with only one $x_1$-variant~$\tau$,
namely the one that makes (\ref{eq:exampleZ3true2}) true.
We of course take $\tau(x_1)$ equal to~$0$ since this is the identity element.

We now need to show that (\ref{eq:exampleZ3true2}) holds for this~$\tau$.
Since $x_2$ is {\em universally} quantified,
we must consider all three $x_2$-variants~$\pi$
of~$\tau$, and show that
\begin{equation}\label{eq:exampleZ3true3}
\mathcal Z_3 \vDash (x_1\cdot x_2 = x_2) [\pi ]
\end{equation}
holds for all three. For instance, we can set $\pi(x_2) = 1$. To evaluate the
truth of~(\ref{eq:exampleZ3true3}), we find that $\pi(x_1\cdot x_2) = 1$
and $\pi (x_2)=1$. Therefore, by the definition of truth for the atomic
formula $x_1\cdot x_2 = x_2$, we have that (\ref{eq:exampleZ3true3}) is true
when $\pi(x_2)=1$.  Similar arguments work when $\pi(x_2)=0$ and $\pi(x_2)=2$.
\end{example}

\begin{example}\label{example:Z3false}
Now consider the sentence~$B$ equal to $\exists x_1\, \forall x_2\, (x_1\cdot x_2 = x_1)$.
We claim that
\[
\mathcal Z_3 \nvDash \exists x_1\, \forall x_2\, (x_1\cdot x_2 = x_1) [\sigma]
\]
for any object assignment~$\sigma$.
Since $B$~is a sentence, we can also write this as $\mathcal Z_3 \nvDash B$, without needing
to mention~$\sigma$.
To prove this, we must show that
\begin{equation}\label{eq:exampleZ3false2}
\mathcal Z_3 \nvDash \forall x_2 \, (x_1\cdot x_2 = x_1) [\tau ]
\end{equation}
is false for every $x_1$-variant~$\tau$ of~$\sigma$. There are
three $x_1$-variants~$\tau$ that must be considered, namely with
$\tau(x_1)=0$, $\tau(x_1)=1$ and $\tau(x_1)=2$. We do just
$\tau(x_1)=0$; the other two cases are similar.

To prove that (\ref{eq:exampleZ3false2}) is false for this~$\tau$,
we must find a $x_2$-variant~$\pi$ of~$\tau$ such that
\begin{equation}\label{eq:exampleZ3false3}
\mathcal Z_3 \vDash (x_1\cdot x_2 = x_1)[\pi]
\end{equation}
is false. It is enough to find one such~$\pi$ since
$x_2$ is {\em universally} quantified. We can take $\pi(x_2)=1$ for instance.
Now, $\pi(x_1\cdot x_2) = 1$ and $\pi(x_1)=0$. Therefore, by
the definition of truth for the atomic formula
$x_1\cdot x_2 = x_1$, we have that (\ref{eq:exampleZ3true3})
is false, as desired.
\end{example}
