\section{Compactness Theorem}\label{sec:compactnessFO}
\index{Compactness Theorem|(}

The Compactness Theorem states
that $\Gamma \vDash A$ if and only if there is a finite subset~$\Gamma_0$
of~$\Gamma$ such that $\Gamma_0 \vDash A$:
\begin{theorem}[Compactness Theorem for Sentences]\label{thm:CompactnessSentencesFO}
Let $\Gamma$ be a set of sentences and $A$ be a formula.
\begin{itemize}
\item[\rm (a)] $\Gamma$ is satisfiable if and only if $\Gamma$ is finitely satisfiable.
\item[\rm (b)] $\Gamma \vDash A$ if and only if there is a finite subset~$\Gamma_0$ of~$\Gamma$
such that $\Gamma_0 \vDash A$.
\end{itemize}
\end{theorem}
\begin{proof}
This follows immediately
from parts~(a) of the Soundness and Completeness Theorems.
Namely, suppose $\Gamma \vDash A$. Then $\Gamma \vdash A$ by the Completeness
Theorem, and thus, by the finiteness of $\FO$-proofs, there is a finite subset~$\Gamma_0$
of~$\Gamma$ such that $\Gamma_0 \vdash A$. Finally, the Soundness Theorem implies
that $\Gamma_0\vDash A$.
\end{proof}

The Compactness Theorem also holds when $\Gamma$~is a set of formulas.

\begin{corollary}[Compactness Theorem for Formulas]\label{thm:CompactnessFO}
Let $\Gamma$ be a set of formulas and $A$ be a formula.
\begin{itemize}
\item[\rm (a)] $\Gamma$ is satisfiable if and only if $\Gamma$ is finitely satisfiable.
\item[\rm (b)] $\Gamma \vDash A$ if and only if there is a finite subset~$\Gamma_0$ of~$\Gamma$
such that $\Gamma_0 \vDash A$.
\end{itemize}
\end{corollary}

\begin{proof}
This follows immediately from the Compactness Theorem
for sentences with the aid of Theorem~\ref{thm:variablesToConstants}
about replacing variables with new constant symbols for
semantic implication.
Let $A$ and $\Gamma$ use the language~$L$. Let $L^\prime$
be obtained by adding new constant symbols $d_1, d_2, d_3,\ldots$.
Then by Theorem~\ref{thm:variablesToConstants},
$\Gamma$~is satisfiable if and only if $\Gamma(\vec d/\vec x)$ is
satisfiable. Likewise,
$\Gamma$~is finitely satisfiable if and only if $\Gamma(\vec d/\vec x)$ is
finitely satisfiable. And, by Theorem~\ref{thm:CompactnessSentencesFO},
$\Gamma(\vec d/\vec x)$ is
satisfiable if and only if it is finitely satisfiable. That proves part~(a).
Part~(b) is proved similarly.
\end{proof}


\subsection*{Applications of the Compactness Theorem.}

\paragraph*{Finiteness and overspill.}
Our first application of the Compactness Theorem
is about the undefinability of finiteness. We now again work with
an arbitrary fixed language~$L$ and assume that it contains
the equality sign.\footnote{It is natural to assume the
presence of the equality sign,~$=$, when talking
about the finiteness of models. However, with some difficulty,
the results on
the undefinability of finiteness can be
reformulated to apply to languages without equality.}

\begin{definition}
A structure~$\mathfrakA$ is \idxem{finite} if its universe~$|\mathfrakA|$ is finite.
The \idxem{cardinality} of~$\mathfrakA$ is the cardinality of~$|\mathfrakA|$.
\end{definition}

\begin{theorem}\label{thm:finiteNonDefine}
There is no sentence~$A$ that defines the set of finite structures.
In other words, the class of finite structures is not an \idxnm{elementary class} ($\EC$).
\end{theorem}

\begin{proof}
Suppose, for the sake of a contradiction,
that there is a sentence~$A$ such that, for all structures~$\mathfrakA$,
$\mathfrakA \vDash A$ holds if and only if $\mathfrakA$~is finite.
From Exercise~\ref{exer:atleastK}, for any fixed $k>1$,
there is a formula  $\TAtLeast_k$ which states that $|\mathfrakA|$ has cardinality
at least~$k$.  For example, one way to construct
$\TAtLeast_k$\index{at least ($\TAtLeast_k)$} is to form
the sentence $\exists x_1\,\cdots\,\exists x_k \bigand_{i<j} x_i \neq x_j$.

Let $\Gamma$ be the set $\{ \TAtLeast_k : k\ge 2 \}$. Clearly, $\mathfrakA \vDash \Gamma$
holds if and only if $\mathfrakA$~is infinite. Therefore,
$\Gamma \cup \{ A \}$ is an unsatisfiable set of sentences, since any model of
$\Gamma \cup \{ A \}$ would have to be both finite and infinite.
By the Compactness Theorem, there is a finite subset $\Gamma^\prime$ of~$\Gamma$
such that $\Gamma^\prime \cup \{ A \}$ is unsatisfiable.

Let $k$ be the maximum value such that $\TAtLeast_k$ is in~$\Gamma^\prime$.
Let $\mathfrakA$ be any structure of size at least~$k$. There must exist such
a structure~$\mathfrakA$ since it can be formed by letting the universe
have $k$~objects and defining the interpretations of the non-logical
symbols arbitrarily. But then, $\mathfrakA \vDash A$ since it is finite,
and $\mathfrakA$ has size~$\ge k$. Therefore, $\mathfrakA$ satisfies
every sentence in~$\Gamma^\prime \cup \{ A \}$, contradicting the unsatisfiability
of~$\Gamma^\prime \cup \{ A \}$.
\end{proof}

The set~$\Gamma$ from the proof defines the class
of infinite structures, since $\mathfrakA \vDash\penalty10000 \Gamma$
holds if and only if $A$~is infinite.  In the terminology
of Section~\ref{sec:definability}, this means
that the class of infinite structures is an
\idxnm{elementary class in the wide sense}~($\EC_\Delta$),
and is equal to $\TMod{\Gamma}$.

We can now state an even stronger form
of Theorem~\ref{thm:finiteNonDefine} that states that there is not
even an infinite set of sentences that defines the property of
being finite.

\begin{theorem}\label{thm:finiteNotEcDelta}
The class of finite structures is not $\EC_\Delta$. That is,
there is no set~$\Pi$ of sentences such that, for all structures~$\mathfrakA$,
$\mathfrakA\vDash \Pi$ if and only if $\mathfrakA$~is finite.
\end{theorem}

Theorem~\ref{thm:finiteNotEcDelta} is an immediate corollary
of Theorem~\ref{thm:finiteNonDefine} and the next theorem.

\begin{theorem}\label{thm:EcDeltaAndCo}
Let $\mathcal S$ be a class of $L$-structures
and $\mathcal T$~be the complement of~$\mathcal S$, namely $\mathcal T$
is the class of $L$-structures which are not in~$\mathcal S$.
Suppose both $\mathcal S$ and~$\mathcal T$ are~$\EC_\Delta$.
Then both $\mathcal S$ and~$\mathcal T$ are~$\EC$.
\end{theorem}

\begin{proof}
Suppose $\mathcal S = \TMod \Gamma$ and $\mathcal T = \TMod \Pi$;
in other words, $\mathfrakA \in \mathcal S$ if and only if $\mathfrakA \vDash\Gamma$,
and $\mathfrakA \in \mathcal T$ if and only if $\mathfrakA \vDash\Pi$.
Since $\mathcal S \cap \mathcal T$ is empty,
$\Gamma \cup \Pi$ is inconsistent.  By the Compactness Theorem,
there is a finite subset of $\Gamma \cup \Pi$ which is inconsistent.
Express this inconsistent subset as $\Gamma^\prime \cup \Pi^\prime$
where $\Gamma^\prime \subseteq \Gamma$ and $\Pi^\prime \subseteq \Pi$.
Let $A$~be the sentence $\bigand \Gamma^\prime$ and $B$~be
the sentence $\bigand \Pi^\prime$.

Suppose $\mathfrakA \vDash A$, or equivalently that $\mathfrakA \vDash \Gamma^\prime$.
By the unsatisfiability
of $\Gamma^\prime \cup \Pi^\prime$, we have $\mathfrakA \nvDash \Pi$.
Therefore $\mathfrakA \notin \mathcal T$; hence $\mathfrakA \in \mathcal S$.
On the other hand, suppose $\mathfrakA \nvDash A$. Then $\mathfrakA \nvDash \Gamma$,
so $\mathfrakA \notin \mathcal S$. This shows
that $\mathfrakA \vDash A$ if and only if $\mathfrakA \in \mathcal S$.
In other words, $\mathcal S = \TMod A$.
A similar argument shows $\mathcal T = \TMod B$.
Therefore, $\mathcal S$ and $\mathcal T$ are elementary classes ($\EC$).
\end{proof}

The next theorem states that a theory with arbitrarily large models
has an infinite model.  This is often called an ``\idxnm{overspill}''
theorem because it states that a property about
arbitrarily large finite cardinalities
spills over to infinite cardinalities. It will
come up again later as a form of the
L\"owenheim-Skolem\index{Lowenheim Skolem Theorem@{L\"owenheim-Skolem Theorem}}
in Section~\ref{sec:LowenheimSkolem}.

\begin{theorem}\label{thm:overspillSatisfiabilty}
Let $\Gamma$~be a set of sentences
and suppose that, for every $k\in \mathbb N$, $\Gamma$~has a model
of cardinality~$\ge k$.  Then $\Gamma$ has an infinite model.
\end{theorem}

\begin{proof}
Let $\Pi$ be the set of sentences
$\Gamma \cup\penalty10000 \{ \TAtLeast_k :\penalty10000 k\ge\penalty10000 1 \}$.
Since $\Gamma$ has models of cardinality $\ge k$ for all~$k$,
$\Pi$~is finitely satisfiable.
Hence, by the Compactness Theorem, $\Pi$~is satisfiable. Any model
of~$\Pi$ is a model of~$\Gamma$ of course. In addition, it must have
cardinality $\ge k$ for all integers~$k$. Hence it must be an infinite
model of~$\Gamma$.
\end{proof}

\paragraph*{A nonstandard model of the theory of the integers.}
\index{integers, theory of}

The theory of the integers is equal
to $\TTh {\mathcal N}$, namely the set of
sentences true in $\mathcal N = ( \mathbb N, 0, S, +, \cdot)$.
By the remark after Definition~\ref{def:theoryStructure},
$\TTh {\mathcal N}$~is complete. By definition, it fully describes
all first-order truths of the integers. Nonetheless, it does not
uniquely characterize the integers.

\begin{theorem}\label{thm:nonstdIntegers}
Let $L_\PA = \{0,S, +, \cdot\}$ be the language of~$\mathcal N$.
There is an $L_\PA$-structure~$\mathfrakA$ which is elementarily
equivalent to~$\mathcal N$ but not isomorphic to~$\mathcal N$.
\end{theorem}

Such a structure~$\mathfrakA$ is called a \idxem{nonstandard model}
of the integers.
Recall that $\mathfrakA$ and~$\mathcal N$ being elementarily equivalent means that
$\TTh \mathfrakA = \TTh {\mathcal N}$.  We haven't yet written out the
precise definition of isomorphic (for this see
Definition~\ref{def:isomorphic}), but it means what one would expect;
namely, that there is a \idxnm{bijection} of the two universes that preserves
the meanings of the constants, predicates, and functions.

\begin{proof}
Let $c$ be a new constant symbol. Let $L^\prime = L_\PA \cup \{c\}$.
For each $i\in \mathbb N$, let $\underline i$ denote
the term $S(S(\cdots S(S(0)) \cdots ))$ with $i$~occurrences
of~$S$. For instance, $\underline 0$ is the term~0, and $\underline 1$ is the term~$S(0)$,
and $\underline 2$ is  the term~$S(S(0))$, etc. Thus $\underline i$~is
a closed term, and in~$\mathcal N$ it denotes the integer~$i$.
In other words, ${\underline i}^{\mathcal N}$ is equal to~$i$.
The terms $\underline i$ are called \emph{numerals}\index{numeral}.

Let $\Gamma$~be the set of sentences
$\TTh {\mathcal N} \cup \{ c \neq \underline i : i\ge 0 \}$.
It is not hard to see that
every finite subset of~$\Gamma$ is consistent. To see this,
consider the set of sentences~$\Gamma_n$ defined by
$\Gamma_n = \TTh {\mathcal N} \cup \{ c \neq \underline i : 0 \le i < n \}$.
Then $\Gamma_n$ is satisfiable, since we can expand~$\mathcal N$ to
the language~$L^\prime$ by setting the interpretation~$c^{\mathcal N}$
equal to the object~$n \in \mathbb N$. Every finite subset of~$\Gamma$ is
a subset of some~$\Gamma_n$; hence $\Gamma$ is finitely satisfiable. By compactness,
$\Gamma$~is also satisfied by some structure~$\mathfrakA$.

By construction, $\mathfrakA \vDash \TTh {\mathcal N}$. Let $\mathfrakB$
be the restriction of~$\mathfrakA$ to the langauge~$L_\PA$.
By Theorem~\ref{thm:truthInExpansion}, $\mathfrakB \vDash \TTh{\mathcal N}$.
In $|\mathfrakB|=|\mathfrakA|$,
there is an object, namely~$c^\mathfrakA$ that is
distinct from~$\underline i^\mathfrakA$ for all $i \in \mathbb N$.
However, in~$\mathcal N$, every object is equal to
some~$\underline i^{\mathcal N}$. (Note that $\underline i^{\mathcal N}$
is the same as~$i$.) Therefore, $\mathcal N$ and~$\mathfrakB$
are not isomorphic. This is because any
isomorphism $\pi:\mathcal N \rightarrow \mathfrakB$ has to
map each $i = i^{\mathcal N}$ to
$\underline i ^\mathfrakA = \underline i ^\mathfrakB$, but then
$c^\mathfrakA$ is not in the range of~$\pi$.
\end{proof}

The existence of a nonstandard model of the integers means
that it is not possible to give a set of axioms that
are true for the integers and false for any other (nonisomorphic)
structure. This foreshadows the G{\"o}del \idxnm{Incompleteness Theorems}, which
state that there is no effective way to give a set of axioms
that imply all statements true in the integers. That, however, is a much deeper
and more important result than the existence of nonstandard models;
it will be stated properly and proved in subsequent chapters.

\paragraph*{Torsion-free groups.}\index{torsion-free}
The class of torsion-free groups was earlier shown to $\EC_\Delta$
as the theory of groups augmented with the axioms
\[
T_k ~:=~ \forall x(x\neq 1 \limplies x^k \neq 1)
\]
axiomatizes the torsion-free groups.

\begin{theorem}\label{thm:torsionFreeNotEC}
The class of \idxnm{torsion-free} groups is not an elementary class.
\end{theorem}

\begin{proof}
Suppose the class of torsion-free groups is~$\EC$,
so it is equal to $\TMod A$ for some sentence~$A$.
Let $\Gamma$ be the set containing the three group axioms
and the sentences~$T_k$. Then, $A\vDash \Gamma$
and $\Gamma \vDash A$. By compactness, there
is a finite~$\Gamma_0 \subseteq \Gamma$ such
that $\Gamma_0 \vDash A$. Let the $T_k$'s
that appear in~$\Gamma_0$ be
$T_{k_1},\ldots T_{k_n}$. Choose a prime~$p$ that does not
divide any of $k_1, \ldots k_n$, and let $\mathfrakA$ be
a cyclic group of order~$p$. Then $\mathfrakA \vDash \Gamma_0$,
but since it is not torsion-free, $\mathfrakA \nvDash A$.
This is a contradiction.
\end{proof}
