\section{Introduction to First-Order Logic}\label{sec:examplesFO}

First-order logic\index{first-order logic} is a very expressive and powerful formal system,
capable of formalizing statements in a wide range of topics. It is particularly
well suited to the formalization of mathematical statements, but it also works
well in many applications governed by well-defined properties and rules.
First-order logic augments
propositional logic by having variables that range over some
domain of ``objects''\index{object} or ``individuals''. These variables can
be quantified with 
universal~($\forall$) quantifiers\index{univeral quantifier@{univeral quantifier ($\forall$)}}
and existential~($\exists$)\index{existential quantifier@{existential quantifier ($\exists$)}}
quantifiers, so that a formula can express properties of the entire domain
of objects. First-order logic includes ``predicates''\index{predicate symbol} or ``relations'' that
describe properties of individuals or relationships between individuals.
It also includes ``constants''\index{constant symbol} that name individuals,
and ``functions''\index{function symbol} that map individuals to individuals.
First-order logic incorporates propositional logic so that all the constructions
from the previous two chapters are still relevant for first-order logic.

The name ``first-order'' refers to the fact that the quantifiers act
on individuals.  Second-order logic and higher-order logic allow quantifying
over sets of individuals or over functions acting on individuals. Higher-order logics
are ostensibly stronger than first-order logic. On the other hand, practically
any formal logic can be recast as a first-order logic (albeit perhaps in an unnatural manner).
This makes first-order logic a kind
of ``ultimate'' logic; furthermore, first-order logic has very nice properties, including having
a proof system with the Soundness and Completeness Theorems.  In addition, \idxnm{set theory}
is usually formalized as a first-order logic, and type theories can be recast as first-order logics
(at least in principle). Arguably, \idxnm{set theory} and type theories can formalize all of mathematics
as presently practiced,
so this means that all of mathematics can be formalized using first-order logic.

We start by presenting three small examples to illustrate the
essential ingredients of first-order formulas. After that, Section~\ref{sec:syntaxFO}
will give the formal definition of first-order formulas.

\paragraph*{Natural language example.}\index{natural language}
Let's start with the sentence
\begin{quote}
\em Everyone has read a book they don't like.
\end{quote}
This can be expressed as the first-order formula
\begin{equation}\label{eq:PersonNotLikeBook}
\forall x\, (\TPerson(x) \limplies \exists y\, [ \TBook(y) \land \TRead(x,y) \land \lnot \TLikes(x,y) ]).
\end{equation}
using ``predicate symbols''\index{predicate symbol} (also called ``relation symbols'') with the meanings:
\begin{center}
\begin{tabular}{c@{~means~}c}
$\TPerson(x)$ &``$x$ is a person'' \\
$\TBook(y)$ &``$y$ is a book'' \\
$\TRead(x,y)$ & ``$x$ has read $y$'' \\
$\TLikes(x,y)$ &``$x$ likes $y$''.
\end{tabular}
\end{center}
These predicates take individuals as inputs and output true/false values.
The quantifiers
``$\forall x$'' and ``$\exists y$'' mean ``for all~$x$'' and ``for some $y$'',
of course. We are being deliberately vague about the universe of
objects that the quantified variables range over, but it should include,
say, the universe of all people and all objects.

For a second example, consider
\begin{quote}
\em Everyone's mother likes {\em Moby Dick}.
\end{quote}
This can be expressed in first-order logic as:
\begin{equation}\label{eq:MotherMobyDick}
\forall x [\TPerson(x) \limplies \TLikes (\TMother(x), \hbox{\em Moby Dick})].
\end{equation}
Here ``$\TMother$ is a ``function symbol''\index{function symbol}, giving the mother of~$x$
as a function of~$x$. This makes the implicit assumption
that everyone has exactly one mother.  The ``{\em Moby Dick}''
is a ``\idxnm{constant symbol}''; namely it refers to a specific object.

For a third example, consider
\begin{quote}
\em
No two people have read exactly the same books.
\end{quote}
We can express this as
\begin{eqnarray}
\nonumber
\lefteqn{\forall x\, \forall w\, [ \TPerson(x) \land \TPerson(w) \land \lnot x=w } \\
\label{eq:NotSameBooks}
&& \qquad \limplies \exists y (\TBook(y) \land \lnot (\TRead(x,y)\liff \TRead(w,y))) ].
\end{eqnarray}
This introduces the equality predicate,~$=$\index{equality sign ($=$)}.
It is common to write ``$x \neq y$''\index{inequality ($\neq$)}
as an abbreviation for ``$\lnot x=y$''. As a final example, consider
\begin{quote}
\em No one likes a book unless they have read it.
\end{quote}
This can be expressed as:
\begin{equation}\label{eq:LikesRead_1}
\forall x\, \forall z\, [\TPerson(x) \land \TBook(z) \land \TLikes(x,z) \limplies \TRead(x,z)]
\end{equation}
or as
\begin{equation}\label{eq:LikesRead_2}
\lnot \exists x\, \exists z\, [ \TPerson(x) \land \TBook(z) \land \TLikes(x,z) \land \lnot \TRead(x,z) ].
\end{equation}

The formulas (\ref{eq:PersonNotLikeBook}) and~(\ref{eq:MotherMobyDick})
both use the construction
\[
\hbox{%
``$\forall x (\TPerson(x) \limplies\penalty10000 \cdots)$''.}
\]
This is
a common construction used to state that every person~$x$ satisfies the property
in the ellipses~``$\cdots$''. In other words, if we let $\TPersons$ denote the
set of all people, then ``$\forall x (\TPerson(x) \limplies \cdots)$'' means
the same thing as ``$\forall x\in \TPersons\,(\cdots)$''.  The dual construction for
existential quantification is illustrated in formulas
(\ref{eq:PersonNotLikeBook}) and~(\ref{eq:NotSameBooks}): these
two formulas use the construction ``$\exists y ( \TBook(y) \land \cdots)$''
to mean the same thing as ``$\exists y \in \TBooks\, (\cdots)$'' where $\TBooks$
is the set of all books.\footnote{The
notations ``$\forall x \in$'' and ``$\exists x \in$'' will not be included in the
syntax for first-order formulas. As just discussed, this does not cause any loss
of expressiveness for first-order formulas.}

These two constructions are also illustrated in the formulas
(\ref{eq:LikesRead_1}) and~(\ref{eq:LikesRead_2}). The former
uses the pattern ``$\forall \cdots \limplies \cdots$'' and the latter uses
the pattern ``$\exists \cdots \land \cdots$''.  The correspondence
between (\ref{eq:LikesRead_1}) and~(\ref{eq:LikesRead_2}) can
be understood by noting that ``$\forall x$'' means the same
as ``$\lnot \exists x \lnot$''.\label{page:forall_notexistsnot} To state this in English, a
property holds for all~$x$ if and only if it is not the case that
the property fails for some~$x$. If the ``$\forall x$'' and ``$\forall y$''
of~(\ref{eq:LikesRead_1}) are replaced by ``$\lnot \exists x \lnot$''
and ``$\lnot \exists y \lnot$'', then tautological equivalences show
that the result is equivalent to~(\ref{eq:LikesRead_2}). For more
on this, see Section~\ref{sec:implication} below.

The above examples illustrate some of the features of first-order logic,
but they also show that first-order logic is not good at capturing
many of the nuances of natural language. For example, it is presumed that for every
$x$ and~$y$, the predicate $\TRead(x,y)$ is either true or false.
There is no consideration of whether $x$ might have read only part of~$y$, or just
skimmed it, etc. Likewise, the constant ``{\em Moby Dick}'' is presumed to
denote one particular book; there is no allowance for nuance about different editions
of the book, etc. In the same vein, it is presumed that everyone has a
unique mother: something which is true in most cases, but of course is
not universally true. Finally, there is presumed to be a definite, fixed universe
of objects that quantified variables range over; this can also be
a problematic assumption in the setting of natural language.
\bigskip

First-order logic is better at capturing mathematical statements, as the next
examples show.

\paragraph*{Group theory examples.}\index{group theory|(}
A group is a mathematical structure with a binary operation that
is associative and has an identity element and inverses. To formalize groups,
we first need to choose a first-order language, namely the predicate symbols,
function symbols, and constant symbols that may be used in formulas.  For (multiplicative)
groups, we can use the language with three symbols: a binary
function $\cdot$ for multiplication, a unary inverse function $^{-1}$, and a
constant symbol~$1$ for the identity element. As is usual, the equality predicate,~$=$,
is also allowed. The three axioms for groups
can be written as:\footnote{When we formally define first-order logic in the next section,
we will require all function symbols to be written
in prefix notation. Thus, strictly speaking, we should write $\cdot(x,y)$
and ${}^{-1}(x)$ instead of $x \cdot y$ and~$x^{-1}$. However, for readability, we use the
more common notations $x \cdot y$
and $x^{-1}$ as abbreviations for terms written in prefix notation.}
\begin{equation}\label{eq:GroupAxioms}
\hbox{\begin{tabular}{cl}
$\forall x\, \forall y\, \forall z\, [ x\cdot (y\cdot z) = (x \cdot y) \cdot z ]$ & - Associativity \\[0.3ex]
$\forall x\, ( 1\cdot x = x \land x \cdot 1 = x)$ & - Identity\index{group identity} \\[0.3ex]
$\forall x\, ( x \cdot x^{-1} = 1 \land x^{-1} \cdot x = 1 )$ & - Inverses\index{group inverse}
\end{tabular}
}  % end of \hbox
\end{equation}

Of course, there are many possible groups. However, these three axioms exactly characterize
groups in the sense that a mathematical structure is a group if and only if it satisfies
these three axioms.

Other languages for groups can be used instead of $\cdot$, ${}^{-1}$ and~$1$. For instance,
it is also possible to use the language containing only the binary group operation~$\cdot$. This would not
reduce the expressibility of formulas. Indeed, the identity and inverse axioms can be rewritten
to use just the function symbol~$\cdot$ as:
\begin{center}
\begin{tabular}{cl}
$\exists z\, \forall x\, ( x \cdot z = x)$ & - Right identity\index{group identity} \\[0.3ex]
$\forall x\, \exists y\, \forall u\, [ u \cdot (x \cdot y) = u ]$ & - Right inverses\index{group inverse}
\end{tabular}
\end{center}
To keep things simpler, these axioms state only the existence of a right identity
element and of right inverses; however, it is easy to modify them to give the full identity
and inverses axioms.\footnote{And, in any event, it is well known that right identity and right inverses
axioms (together with associativity) imply the full identity and inverses axioms. In addition,
it is easy to see that the right inverses axiom is stated in a way that implies the
right identity axiom.}

Many group properties can be expressed in first-order logic. For instance,
$\forall x\,(x\neq 1 \limplies x\cdot x \neq 1)$ states that no
element has order\index{order (group element)} two. Similarly,
$\forall x\,(x\neq\penalty10000 1 \limplies\penalty10000  x\cdot\penalty10000(x\cdot\penalty10000 x) \neq\penalty10000 1)$ 
states that no
element has order three. In the same way, for
any $k\ge 2$, there is a formula $T_k$ expressing that no element has
order~$k$. Specifically, let $k\ge 2$ be a fixed integer. Let $x^k$ denote the
$k$-fold product of $x$ with itself, and let $T_k$~be the formula
$\forall x(x\neq 1 \limplies x^k \neq 1)$.
The formula~$T_k$ states that the order of~$x$ does not divide~$k$.
Let $\TDiv(k)$ denote the (finite) set of
proper divisors of~$k$.
Then $T_k(x) \land x^k = 1 \land \bigand_{i\in \TDiv(k)} \lnot T_i(x)$ 
expresses that $x$~has order exactly~$k$.

As another example, the \idxnm{center} of a group can be defined
by the formula $\forall x\, (z\cdot x = x \cdot z)$, as this characterizes
the members~$z$ of the center. Note how $z$ is not quantified in this. This
is because the formula expresses a property of the object~$z$.

However, there are many common concepts in group theory that are not
first-order expressible. For instance, first-order logic is unable
to talk directly about subgroups since variables range only over group
elements and there is no way to existentially or universally quantify
over subgroups. Certainly, special subgroups (such as the center) can be defined,
but properties
of arbitrary subgroups cannot in general be expressed in first-order logic.

A group is called \idxem{torsion-free} if there are no elements of finite order
(other than the identity). The infinite set of formulas
\[
\Gamma ~=~ \{ T_2, T_3, T_4, \ldots \}
\]
expresses the property of being torsion-free. However, as we
see later in Section~\ref{sec:compactnessFO} no single
formula, nor finite set of formulas, characterizes the property of a group
being torsion-free. It will follow that there is no set~$\Pi$
of first-order formulas, finite or infinite,
that characterizes the property of {\em not} being torsion-free. For this, see
Theorem~\ref{thm:torsionFreeNotEC}.
\index{group theory|)}

\paragraph*{The theory of the integers.}\index{integers, theory of|(}
The first-order theory of the integers is of central importance in mathematical
logic, as well as in mathematics in general. For logicians, it is typical to
define the integers $\mathbb N$ as $\{0,1,2,\ldots\}$, namely starting
with~0. So, henceforth, we generally use ``integer''\index{integers} 
to mean ``nonnegative integer''.

It is traditional for the language of arithmetic to include the
following symbols: 
\begin{itemizeLoose}
    \item  the constant~$0$, 
    \item the unary \idxnm{successor} function~$S$, and 
    \item the binary addition and multiplication functions $+$ and~$\cdot$.
\end{itemizeLoose}
These have their usual meanings; the successor function $S(x)$ is intended
to denote $S(x) = x+1$.
As usual, the equality symbol~$=$ is also allowed. Many properties of integers
can be expressed with these symbols. For instance, the constants $1$ and~$2$
can be represented by the terms $S(0)$ and $S(S(0))$.  And, the relation ``$x \le y$''
is expressible with
\begin{equation}\label{eq:xLEy}
\exists z\, ( x+z = y ).
\end{equation}
Note this formula has two ``free''\index{free variable} variables~$x, y$ (that is, $x$ and~$y$
are not quantified), and one ``bound''\index{bound variable} (quantified) variable~$z$. This means
that the formula expresses
a property of $x$ and~$y$.

The property ``$x$ is a divisor of~$y$''
can be expressed as $\exists z (z \cdot x = y)$.
The property that ``$x$~is even'' can be expressed as either
\[
\exists v\,( v + v = x ) \qquad \hbox{or} \qquad
\exists z\,( z \cdot S(S(0)) = x.
\]
Then, ``$x$ is a power of~$2$'' can expressed by saying that
any divisor of~$x$
either is equal to~1 or is even; namely,
\[
\forall u\,(\exists z\,(z\cdot u = x) \limplies u=S(0) \lor \exists v\,( v+v = u)).
\]
The property that $x$ is prime\index{prime number} can be expressed by
\begin{equation}\label{eq:xPrime}
x\neq S(0) \land \forall y\, (\exists z\, (y\cdot z=x) \limplies y=S(0) \lor y=x).
\end{equation}
This expresses that any divisor~$y$ of~$x$ is equal to either
$1$ or~$x$.

Let $\TPrime(x)$ denote the formula~(\ref{eq:xPrime}).
The property that there exist infinitely many
primes can be expressed by
\begin{equation}\label{eq:infinityPrimes}
\forall x\, \exists u\, ( x \le u \land \TPrime(u) ).
\end{equation}
Here $\TPrime(u)$ means the result of substituting $u$
for each occurrence of~$x$ in $\TPrime(x)$.\footnote{A full
treatment of substitution is quite complicated, and will be discussed
in detail in Section~\ref{sec:substitution}.
There the formula $\TPrime(y/x)$ will mean the result of substituting
$y$ for~$x$ in $\TPrime$, and the notation $\TPrime(y)$ will be
a convenient informal notation.} The formula~(\ref{eq:infinityPrimes}) does not
say directly that there are infinitely many primes, as the property of
infinitude cannot be directly expressed in first-order logic.
Instead, (\ref{eq:infinityPrimes}) states that
there are arbitrarily large primes.

In contrast to group theory, where mathematicians study many, very different,
groups, there is only one set of integers. As is shown later,
first-order logic is very expressive
for the integers. Nonetheless, there are inherent difficulties in
expressing, axiomatizing, and proving some important concepts about the integers.
In fact, it is impossible to rule out ``nonstandard integers\index{nonstandard integers}''
which satisfy exactly the same first-order properties as the integers. It is also impossible
to give a decidable set of axioms that implies exactly the true first-order statements
about the integers.  These limitations
will be established later, in the context of the Completeness and Compactness
Theorems and in the context of undecidability and
the G{\"o}del Incompleteness Theorems.
\index{integers, theory of|)}

\paragraph*{The theory of directed graphs.}
A directed \idxnm{graph}\index{graph!directed}~$G = (V,E)$ 
consists of a nonempty set~$V$ of vertices and a
binary edge relation~$E(\cdot, \cdot)$.  If $x,y$ are vertices, then
the condition $E(x,y)$ indicates that there is a (directed) edge from
$x$ to~$y$. Various simple properties of directed graphs can be 
expressed in first order logic.  These include statements such as:
\begin{itemizeLoose}
\item The condition that $G$ is complete graph\index{graph!complete} is expressed by
the sentence $\forall x \forall y E(x,y)$.
\item The condition that $x$ is a \idxnm{sink} vertex is expressed by
$\forall y \lnot E(x,y)$.
\item The condition that $G$ is \idxnm{loop-free} is expressed by the sentence
$\forall x \, \lnot E(x, x)$.
\item The condition that $G$ contains a (directed) triangle can be expressed
with the sentence
\[ 
\exists x\, \exists y\, \exists z \, (E(x,y) \land E(y,z) \land E(z,x)).
\]
\end{itemizeLoose}

A common way to handle undirected graphs with first-order logic is to adopt the
convention that undirected \idxnm{graph}\index{graph!undirected} is a directed graph
that satisfies the condition
\begin{equation}\label{eq:undirectedEdges}
\forall x \, \forall y \, [E(x,y) \liff E(y,x)].
\end{equation}
That is, an undirected edge is represented by the presence of directed edges in both directions,
and a missing undirected edge is represented by the absence of both directed edges.

Exercise~\ref{exer:3colorableProp} discussed the property of an undirected graph being 
3-colorable\index{colorable@{3-colorable graph}}. That exercise worked with a 
fixed graph~$G$, and used $3n$~many variables to represent the colors assigned
to the $n=|G|$ many vertices of~$G$. In contracts to propositional logic,
first-order logic can discuss 
graphs with vertex colors without having to fix a particular~$G$. For this, we use
the first-order language with the binary edge relation~$E$ plus three
unary relation symbols $R$, $G$, and $B$. The conditions $R(x)$, $G(x)$, and~$B(x)$
indicate that the vertex~$x$ is assigned the color red, green, and blue (respectively).
A 3-colorable graph is then an undirected graph that satisfies the conditions
\[
\forall x \, ( R(x) \lor G(x) \lor B(x) ).
\]
and
\[
forall x \, [ (\lnot R(x) \lor \lnot G(x)) \land 
              (\lnot R(x) \lor \lnot B(x)) \land
              (\lnot G(x) \lor \lnot B(x)) ] .
\]
The former states that every vertex has at least one color; the latter states that
each vertex is assigned at most one color.

There is no first-order formula that expresses
the property that an undirected graph is 3-colorable, see Exercise~\ref{exer:3colorableUndef}.  
However, there is an infinite set~$\Gamma$ of first-order formulas which expresses the
condition that an undirected graph is 3-colorable. 
For this, see Example~\ref{example:graphsEC} and Exercise~\ref{exer:graphsECdelta3}.
