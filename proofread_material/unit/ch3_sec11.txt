\section{Definability}\label{sec:definability}

\subsection{Definability of structures}

Many mathematical notions such as groups, rings, fields, integral domains,
graphs, partial orders, total orderings, lattices, Boolean algebras, etc.\ can
be defined with first-order axioms. The example of groups\index{group} was discussed
in Section~\ref{sec:examplesFO}.  Letting $L = \{1, \cdot, ()^{-1}\}$, where
$1$~is a constant symbol, $\cdot$ is a binary function symbol, and $()^{-1}$ is a
unary function symbol, then the class of all groups can
be characterized as being the class of $L$-structures that satisfy the
three (first-order) axioms for groups given in Equations (\ref{eq:GroupAxioms}).\footnote{For
readers familiar with set theory: We call it
the {\em class} of groups instead of the {\em set} of all groups, because the class of all groups is not a set.}
Because there are only finitely many axioms for groups,
we call the class of groups an ``\idxnm{elementary class}''.\footnote{In this
context, the term ``\idxnm{elementary}'' means ``first-order''.}

\begin{definition}\label{def:ModClass}
Let $L$ be a language. If $\Gamma$ is a set of $L$-sentences, then
$\TMod \Gamma$, the {\em models of}\index{models@{models of ($\TModBase$)}}~$\Gamma$
is the class of $L$-structures which are models of~$\Gamma$.
If $A$ is a sentence, then $\TMod A$ denotes the class of $L$-structures
which satisfy~$A$, namely $\TMod{\{A\}}$.
\end{definition}

\begin{definition}\label{def:elementaryclass}
Let $L$ be a language and $\mathcal S$~be a class of $L$-structures.
The class~$\mathcal S$ is an \emph{\idxnm{elementary class}}~($\EC$) if there
is an $L$-sentence~$A$ such that $\mathcal S = \TMod A$.
The class~$\mathcal S$ is an \emph{\idxnm{elementary class in the wide sense}}~($\EC_\Delta$) if there
is a set of $L$-sentences~$\Gamma$ such that $\mathcal S = \TMod \Gamma $.
\end{definition}

If $\Gamma$ is finite, then we can let $A$
be the sentence $\bigand \Gamma$, the conjunction of the
members of~$\Gamma$. Therefore, for a finite set~$\Gamma$,
$\TMod \Gamma = \TMod A$ is $\EC$. For instance, the class of groups is $\EC$.

The \idxnm{torsion-free} groups are an example of an
elementary class in the wide sense, namely an $\EC_\Delta$ class.
Recall that $T_k$ was defined to be the sentence
$\forall x(x\neq 1 \limplies x^k \neq 1)$.  Letting
$\Gamma$ be the set of sentences consisting of the three group axioms
and the sentences $T_k$ for $k\ge 2$, the class of torsion-free
groups is equal to $\TMod \Gamma $ and thus is $\EC_\Delta$.
On the other hand, it will be a consequence
of the Compactness Theorem in Section~\ref{sec:compactnessFO} that
the class of torsion-free groups is not~$\EC$ and that
the class of groups that are not torsion-free is not even $\EC_\Delta$.

\begin{example}\label{example:graphsEC}
\index{graph}\index{graph!directed}
We show that directed graphs, and also undirected graphs,
are elementary classes.  Let the language be $L = \{ E \}$ for~$E$
a binary connective.  For directed graphs, we view $E(x,y)$
as meaning that there is an edge from vertex~$x$ to vertex~$y$.
Let $A$ be the sentence $\forall x\, (\lnot E(x,x))$; this expresses
the property that there are no loops in the directed graph.
Then the class of directed graphs (without loops\index{loop-free}) is equal
to $\TMod A$, and hence is $\EC$.

\index{graph!undirected}
The class of undirected graphs uses the same language~$L$.
In an undirected graph, the edge relation must be symmetric\index{symmetry}.
This can be expressed by the sentence~$B$ equal to
$\forall x\, \forall y\, (E(x,y) \liff E(y,x))$. Since the class of
undirected graphs has two axioms $A$ and~$B$, it is $\EC$.

Exercises \ref{exer:graphsECdelta}-\ref{exer:graphsECdelta3} ask you to prove that
the class of infinite graphs, the class of cycle-free graphs, and the class of
3-colorable\index{colorable@{3-colorable graph}} graphs
are elementary in the wide sense~($\EC_\Delta$). These three classes
are not $\EC$.
\end{example}

\begin{example}\label{example:linearorder}
A \idxem{linear order} is a structure over the language $L = \{ < \}$
which satisfies the following three sentences (axioms):

\begin{enumerateLoose}
\item $\forall x\, (\lnot \, x < x)$.  Irreflexivity.\index{irreflexivity}
\item $\forall x\, \forall y\, \forall z\, ( x < y \land y < z \limplies x < z)$. Transitivity.\index{transitivity}
\item $\forall x\, \forall y\, (x=y \lor x < y \lor y < x ) $. Strong connectivity. \index{tricotomy}
\end{enumerateLoose}
A linear order is \idxem{dense}\index{dense linear order} if it additionally satisfies the axiom
\begin{enumerateLoose}[resume]
\item $\forall x\, \forall y\, (x<y \limplies \exists z\, (x< z \land z < y))$.
\end{enumerateLoose}
That is, a linear order is dense if, for every two distinct objects $x$ and~$y$, there
is an object~$z$ strictly between $x$ and~$y$.   Clearly, the class of strict linear
orders and the class of dense linear orders (DLO's) are both $\EC$.
\end{example}

\begin{example}\label{example:field}
A \idxnm{field} is a structure over the language
$0, 1, +, \cdot$ where $0$ and~$1$ are constant symbols and
$+$ and~$\cdot$ are binary function symbols.
The usual axioms for fields are the sentences:
\begin{enumerateLoose}
\item $\forall x\, \forall y\, (x+y = y+x)$.  Addition is commutative.\footnote{The commutativity
of addition follows from (i.e., is logically implied by) the other nine axioms. Thus, this axiom
could be omitted.}
\item $\forall x\, \forall y\, (x+(y+z) = (x+y)+z)$.  Addition is associative.
\item $\forall x\, (x + 0 = x \land 0 + x = x )$.  Additive identity.
\item $\forall x\, \exists y\, (x+y = 0 \land y+x = 0)$. Additive inverses.
\item $\forall x\, \forall y\, (x\cdot y = y\cdot x)$.  Multiplication is commutative.
\item $\forall x\, \forall y\, (x\cdot (y\cdot z) = (x\cdot y)\cdot z)$.
    Multiplication is associative.
\item $\forall x\, (x \cdot  1 = x \land 1\cdot x = x )$.  Multiplicative identity.
\item $\forall x\, (x\neq 0 \limplies \exists y\, (x\cdot y = 1 \land y\cdot x = 1))$.
    Multiplicative inverses.
\item $\forall x\, \forall y\, \forall z\,( (x+y) \cdot z = (x\cdot z) + (y \cdot z) )$. Distributivity.\index{distributivity}
\item $0 \neq 1$.
\end{enumerateLoose}
Thus the class of fields is $\EC$.
\end{example}

\begin{example}\label{example:orderedfield}
An \idxnm{ordered field} is a field that has a strict linear order that
is compatible with the field operations. The language of ordered
fields is $0, 1, +, \cdot, <$. The axioms for ordered fields
consist of the axioms (a)-(c) of strict linear orders from
Example~\ref{example:linearorder}, axioms (a)-(j) for fields
from Example~\ref{example:field}, plus the following two axioms
that connect the order~$<$ to the field operations.
\begin{enumerateLoose}
\item $\forall x\, \forall y\, \forall z\, (x<y \limplies x+z < y+z )$.
\item $\forall x\, \forall y\, (0<x \land 0<y \limplies 0< x\cdot y)$.
\end{enumerateLoose}
Thus the class of ordered fields is $\EC$.
\end{example}

\begin{example}\label{example:RCFaxioms}
A \idxnm{real closed field} is an ordered field that satisfies exactly the same first-order
properties as the field $\mathcal R =(\mathbb R, 0, 1, +, \cdot,<)$ of
\idxnm{real numbers}.  The axioms for real closed fields include all the
axioms for ordered fields, plus the following infinite set of axioms:
\begin{enumerateLoose}
\item Every positive field element has a square root:
\[
\forall x\, (0 < x \limplies \exists y\, ( y\cdot y = x)).
\]
\item Every odd-degree polynomial has a root: namely, if $n$ is odd:
\[
\forall a_0\, \forall a_1 \,\cdots\,  \forall a_n\, ( a_n \neq 0 \limplies
\exists x \, (a_n\cdot x^n + a_{n-1} \cdot x^{n-1} + \cdots + a_1 x + a_0 = 0 )).
\]
Here $x^k$, for $k\in\mathbb N$, means the $k$-fold product of~$x$ with itself.  
(The exponential
function is {\em not} part of the language for real closed fields.)
\end{enumerateLoose}
There are infinitely many axioms for real closed fields of type~(b),
namely one for each odd $n\ge 3$.
Hence the class of real closed fields is $\EC_\Delta$.  (It can
be shown that the class of real closed fields is not~$\EC$.)
\end{example}

The most prominent example of a model of the axioms of real closed fields
is the field $\mathcal R = (\mathbb R, 0, 1, +, \cdot, <)$ of
the real numbers. It is straightforward to see that all the axioms of
real closed fields are satisfied by~$\mathcal R$. More than this
is true: the above axioms for real closed fields are sufficient
to logically imply all sentences that are true in~$\mathbb R$. The next
definitions make this precise.

\begin{definition}\label{def:theory}
A \idxem{theory} over a language~$L$ is set~$T$ of $L$-sentences
which is closed under logical consequence. In other words,
if $A$ is an $L$-sentence and $T\vDash A$, then $A\in T$.
\end{definition}

\begin{example} The theory of groups\index{group} is
the set of sentences that are true in all groups.
Equivalently, the theory of groups is the set of
logical consequences of the three axioms for groups
that were given earlier in (\ref{eq:GroupAxioms}).

The theory $\DLO$ of dense linear orders\index{dense linear order}
is the set of logical consequences of the axioms (a)-(d)
given in Example~(\ref{example:linearorder}).

The theory $\RCF$ of real closed fields\index{real closed field}
is the set of logical consequences of the axioms for real closed fields
presented across Examples (\ref{example:linearorder})-(\ref{example:RCFaxioms}).
\end{example}

The example of $\RCF$ is particularly interesting because it
turns out that the axioms for $\RCF$ completely characterize what is
true in the structure~$\mathbb R$ of real numbers. Indeed, for
any sentence~$A$ in the language of~$\RCF$ either $\RCF \vDash A$ or
$\RCF \vDash \lnot A$.\footnote{This important result was proved
by Tarski~\cite{Tarski:decisionmethod}, but its proof is too complex for
us to present it.} Since $\RCF$ is closed under logical consequence
this means that, for every sentence~$A$, either $A\in \RCF$ or $\lnot A \in \RCF$.
That is to say, $\RCF$ is ``\idxnm{complete}'':

\begin{definition}\label{def:completeFO}
A theory~$T$ (over the language~$L$) is \idxem{complete} if, for every $L$-sentence~$A$,
either $A\in T$ or $\lnot A \in T$.
\end{definition}

\begin{definition}\label{def:theoryStructure}
Let $\mathfrakA$ be an $L$-structure. The \idxem{theory}
of~$\mathfrakA$ is denoted $\TTh \mathfrakA$\index{theory of ($\TThBase$)} and is the set of $L$-sentences
\[
\TTh \mathfrakA ~=~ \{ A : \hbox{$A$~is an $L$-sentence and $\mathfrakA \vDash A$} \}.
\]
\end{definition}

Note that $\TTh \mathfrakA$ must be complete. This is because any sentence~$A$
will be either true or false in~$\mathfrakA$, and thus one
of $A$ or~$\lnot A$ will be in $\TTh \mathfrakA$.

\begin{definition}
Two $L$-structures $\mathfrakA$ and~$\mathfrakB$
are \idxem{elementarily equivalent} if
$\TTh \mathfrakA$ and $\TTh \mathfrakB$ are equal.
We write $\mathfrakA \equiv \mathfrakB$ to denote that
$\mathfrakA$ and~$\mathfrakB$ are elementarily equivalent.
\end{definition}

Unwinding the definitions, $\mathfrakA$ and~$\mathfrakB$
are elementarily equivalent if and only if they satisfy
exactly the same $L$-sentences. The adverb ``elementarily''
is used since ``\idxnm{elementary}'' is sometimes used to
mean ``first-order''.

It is also useful to talk about the theory of a class of structures:
\begin{definition}
Let $\mathcal S$ be a class of $L$-structures.
The \idxem{theory} of~$\mathcal S$
is denoted $\TTh {\mathcal S}$\index{theory of ($\TThBase$)} and
is the set of $L$-sentences true in all members of~$\mathcal S$.
\[
\TTh {\mathcal S} ~=~ \{ A :
     \hbox{$A$~is an $L$-sentence and $\mathfrakA \vDash A$ for all $\mathfrakA \in \mathcal S$} \}.
\]
\end{definition}
For example, if $\mathcal S$ is the class of all groups, then
$\TTh {\mathcal S}$ is the theory of groups\index{group theory}, namely the set
of sentences true in all groups. In general, $\TTh {\mathcal S}$ may not be
complete. For single structure~$\mathfrakA$, $\TTh \mathfrakA$ is the same
as $\TTh {\{\mathfrakA\}}$ and is complete.

\begin{example}
$\TTh {\mathcal R}$ is equal to $\RCF$.  Any two
models of $\RCF$ are elementarily equivalent.
This follows immediately from the fact that $\RCF$ is
complete and that $\mathcal R \vDash \RCF$
so $\RCF \subseteq \TTh {\mathcal R}$.  (The proof
of these assertions is beyond the scope of this book.)
\end{example}

\begin{example}\index{group}
The theory of groups is not complete. For example,
some groups are order two and satisfy
$\forall x\, (x\cdot x = 1)$, whereas
other groups do not satisfy this.
\end{example}

\begin{example}
The theory DLO of dense linear orders\index{dense linear order}
is not complete. For example, consider the following sentence,
which asserts that there is no least member of the order
(i.e., there is no ``\idxnm{left endpoint}'')
\begin{equation}\label{eq:noLeftEndpoint}
\forall x\, \exists y\, ( y<x ).
\end{equation}
This sentence is true in some models of~$\DLO$; for instance,
in the structure $(\mathbb Q, <)$ of the usual ordering on the
rational numbers. On the other hand, it is false in other
models of~$\DLO$. An example is $(\mathbb Q^{\ge 0}, <)$ of the
usual ordering on the nonnegative rational numbers.  It is easy to check that
both $(\mathbb Q, <)$ and $(\mathbb Q^{\ge 0}, <)$ satisfy all the axioms
for dense linear order.
\end{example}

\begin{example}\label{example:DLOwoEndpoints}
\index{dense linear order!without endpoints}
The theory of \emph{dense linear order (DLO) without endpoints} is axiomatized as
DLO plus two axioms asserting there is no minimum or maximum element:
\[
\forall x\, \exists y\, ( y<x ) \qquad \hbox {and } \qquad \forall x\, \exists y\, ( x<y ).
\]
The first of these was already given in Equation~(\ref{eq:noLeftEndpoint}).  The set
of rationals $(\mathbb Q, <)$ is an example of a model of this theory.  It can be shown
that the theory of DLO without endpoints is complete. In fact, any countable
model of the theory DLO without endpoints is isomorphic to $(\mathbb Q, < )$.
\end{example}

\paragraph*{The theory of arrays.}\label{sec:arays}\index{arrays|(}

The theory of arrays is rather different from the above examples; instead of defining
a conventional mathematical structure, it defines a dynamic construction used in
computer programming. As such, the theory of arrays might be a small part of
a theory for proving the correctness of programs.  Indeed, the theory of arrays can be
incorporated into widely used software verification theorem provers.

The theory of arrays deals with
three sorts of objects: arrays, indices, and values. When writing an informal
expression {\tt a[i] = x}, the ``{\tt a}''~is an array, ``{\tt i}''~is an
index, and ``{\tt x}''~is a value. The most natural way to deal with these
three sorts of objects is to use a ``\idxnm{sorted}'' logic where variables
are labeled explicitly as being of different ``sorts'' or ``types''
and as ranging over an appropriate (sub)universe of objects.
In other words, a sorted logic uses a separate universe for each sort of object.
To formalize sorts within our (non-sorted) first-order logic, we introduce three unary predicates
$\TArray$, $\TIndex$ and $\TValue$ and let the formulas
\[
\TArray(z), \qquad \TIndex(z) \quad \hbox{and} \qquad \TValue(z)
\]
denote that $z$ is an array, an index, or a value, respectively. We could
optionally include an axiom that states that every object~$z$ satisfies
exactly one of the three predicates, but it is not strictly speaking necessary.

The theory of arrays uses two function symbols $\TRead$ and $\TWrite$.
The $\TRead$ function takes two inputs: an array~$a$ and an index~$i$.
It returns a value~$x$, which is intended to be the value stored in
the array~$a$ at the location specified by the index~$i$.
The function $\TWrite(a,i,x)$ takes three inputs: an array~$a$, an index~$i$
and a value~$x$. It returns an array~$b$ which is the array obtained from~$a$ by changing
the value at the location indexed by~$i$ to the value~$y$.  Note that the intuitive
action of the function $\TWrite$ is that it makes a copy of the array~$a$, updates
its value at location~$i$ to equal~$x$, and returns the new updated array.
This differs somewhat from the way software programs update an
array value without making a copy of the array, but it still allows reasoning about
the action of read and write operations.

In the literature, the axioms for arrays would typically be expressed by using
(something similar to)
the following implications:
\begin{eqnarray*}
& a:\TArray; i:\TIndex \Rightarrow \TRead(a,i): \TValue \\
& a:\TArray; i:\TIndex; x:\TValue \Rightarrow \TWrite(a,i,x):\TArray \\
& a:\TArray; i:\TIndex; x:\TValue \Rightarrow \TRead(\TWrite(a,i,x),i)=x \\
& a:\TArray; i,j :\TIndex; x:\TValue \Rightarrow i\neq j \limplies \TRead(\TWrite(a,i,x),j)=\TRead(a,j) \\
& a,b:\TArray \Rightarrow \forall i : \TIndex (\TRead(a,i) = \TRead(b,i)) \limplies a=b
\end{eqnarray*}
A notation such as ``$a:\TArray$'' means that $a$ is a variable that ranges over
objects of type ``array''. Similarly, ``$\TRead(a,i): \TValue$'' means that
that $\TRead(a,i)$ is an object of type ``value''. The variables in the implications are
implicitly universally quantified. The semicolons to the expression before the ``$\Rightarrow$''
act as if they are conjunctions.  We do {\em not} adopt these notations;
instead, the above axioms can be reexpressed in the syntax of first-order logic
as:
\begin{eqnarray*}
&& \forall a\, \forall i\, [\TArray(a) \land \TIndex(i) \limplies \TValue (\TRead( a, i) ) ] \\
&& \forall a\, \forall i\, \forall x\, [\TArray(a) \land \TIndex(i) \land \TValue(x) \limplies \TArray (\TWrite( a, i, x ) ) ] \\
&& \forall a\, \forall i\, \forall x\, [\TArray(a) \land \TIndex(i) \land \TValue(x) \limplies \TRead(\TWrite(a,i,x),i)=x] \\
&& \forall a\, \forall i\,\forall j\, \forall x\, [\TArray(a) \land \TIndex(i) \land \TIndex(j) \land \TValue(x) \land i\neq j  \\
&& \qquad\qquad\qquad \limplies \TRead(\TWrite(a,i,x),j)=\TRead(a,j)] \\
&& \forall a\, \forall b\, (\TArray(a) \land \TArray(b) \limplies [\forall i(\TIndex(i)\limplies \TRead(a,i) = \TRead(b,i)) \limplies a=b]
\end{eqnarray*}
$\TArray$, $\TIndex$, and $\TValue$ are unary predicates; $\TRead$ is a
binary function symbol, and $\TWrite$ is a 3-ary function symbol.
The first two axioms specify the sorts of the functions $\TRead$ and $\TWrite$.
The third axiom states that if one writes $x$ to location~$i$ of an array, and reads
from the same location, one gets the value~$x$ back. The fourth axiom states
that if one writes $x$ to location~$i$, it does not change the value stored
at any different location~$j$.

The fifth axiom is called the axiom of \idxnm{extensionality}. It states that if two arrays have
exactly the same contents, then they are actually the same array. By
equality principle of Theorem~\ref{thm:equalityPrincipleSemantic}
the converse implication
will automatically hold; namely, if $a=b$
then $\forall i\, (\TRead(a,i)=\TRead(b,i))$ holds. Thus, the axiom of extensionality
actually states that $\forall i(\TIndex(i)\limplies \TRead(a,i) = \TRead(b,i))$ can
be taken as the {\em definition} of equality.

\subsection{Definability in structures}

We discussed above how a set of axioms can define a theory~$T$,
and thus a class of structures $\TMod T$. For a different notion of
first-order definability, we work with a fixed structure~$\mathfrakA$ and discuss
how objects, relations, or functions over~$\mathfrakA$
can be defined using first-order formulas.  First, a formula with $k$ free variables
defines a $k$-ary relation in the natural way:
\begin{definition}[Definability of a relation in~$\mathfrakA$]\label{def:definesRelation}
Let $\mathfrakA$ be a structure, and let
$B=B(x_1,\ldots,x_k)$ be a first-order formula containing only $x_1,\ldots,x_k$ free.
Then $B$ \emph{defines}\index{defines!a relation} the $k$-ary relation
on $|\mathfrakA|$ equal to
\[
\{ \langle a_1,\ldots,a_k\rangle : \hbox{$a_1,\ldots, a_k\in |\mathfrakA|$ and $\mathfrakA \vDash B(a_1,\ldots,a_k)$} \}.
\]
This relation is said to be \idxem{definable} in~$\mathfrakA$.
\end{definition}

For example, we earlier discussed that the center of a group can be defined
by the formula $\forall x_2\,(x_1\cdot x_2 = x_2\cdot x_1)$. As another
example, the earlier-discussed formula $\TPrime(x_1)$ defines
the set of primes in the structure~$\mathcal N$ of nonnegative integers.

A related notion is the definability of a particular object in~$\mathcal A$. This is the same
as defining a unary relation that has a single member:

\begin{definition}[Definability of an individual in~$\mathfrakA$]
Let $\mathfrakA$ be a structure and let $a\in |\mathfrakA|$.
The object~$a$ is \idxem{definable} in~$\mathfrakA$ if
there is a formula $B(x_1)$ with $x_1$ the only variable free in~$B$ so
that $a$~is the unique member of~$|\mathfrakA|$ such that
$\mathfrakA \vDash B(a)$. In this case, we say that $B$~\emph{defines}\index{defines!an object}
the object~$a$ in~$\mathfrakA$.
\end{definition}

For example, in $\mathcal N = (\mathbb N, 0, S, +, \cdot ) $, the
object~$1$ is defined by $x_1=S(0)$ and also by $\forall x_2\, (x_1\cdot x_2 = x_2)$.

\begin{definition}\label{def:existsUnique}\index{exists unique ($\exists"!$)}
Let $B(x)$ be a formula.
We write $\exists ! x \, B$, namely ``there exists a unique~$x$ such that $B$ holds'',
as an abbreviation for the formula
\[
\exists x\, [B(x) \land \forall y\, (B(y) \limplies y=x)].
\]
\end{definition}

Thus, $B(x)$ defines an object in~$\mathfrakA$ if and only if $\mathfrakA \vDash \exists ! x\, B(x)$.
Finally, a function is said to be definable in~$\mathfrakA$ if and only if its graph is
definable in~$\mathfrakA$:

\begin{definition}[Definability of a function in~$\mathfrakA$]
Let $\mathfrakA$ be a structure, and let
$B=B(x_1,\ldots,x_k,x_{k+1})$ be a first-order formula containing only $x_1,\ldots,x_{k+1}$ free.
Suppose that
\[
\mathfrakA\vDash \forall x_1\ldots \forall x_k \, \exists ! x_{k+1} B(x_1,\ldots,x_k,x_{k+1}) .
\]
Then $B$ \emph{defines}\index{defines!a function} the $k$-ary function~$f$
on $|\mathfrakA|$ such that, for $a_1,\ldots,a_k \in |\mathfrakA|$,
we have $f(a_1,\ldots,a_k)$ is equal to the (unique) $a_{k+1}$ in~$|\mathfrakA|$ such
that $\mathfrakA \vDash B(a_1,\ldots,a_k,a_{k+1})$ holds.  When this holds, we say that $f$ is
\idxem{definable} in~$\mathfrakA$.
\end{definition}

\begin{example}
Let $\mathfrakA$ be a linear order. Thus $\mathfrakA$ is an $L$-structure
for the language $L = \{ < \}$. A non-strict ordering~$\le$ can be defined for~$\mathfrakA$
with the formula
\[
x_1 < x_2 \lor x_1=x_2.
\]
\end{example}


\begin{example}
As usual, let $\mathcal N$ be the nonnegative integers with the language
$0, S, +, \cdot$. The binary relation~$\le$ can be defined by
\[
\exists x_3\, (x_1 + x_3 = x_2) .
\]
This is verified by noting that the condition is indeed equivalent to $x_1\le x_2$.
The relation~$<$ can be defined by $\exists x_3\, (x_1 + S(x_3) = x_2)$.

The integer square root function $x\mapsto \lfloor \sqrt x \rfloor$ can be defined
in~$\mathcal N$ by
\[
x_2 \cdot x_2 \le x_1 \land x_1 < S(x_2) \cdot S(x_2).
\]
\end{example}

This last example illustrates an important principle: if a function,
a predicate, or an object is definable in an $L$-structure~$\mathfrakA$, then we can
effectively augment the language~$L$ to include that
definable function, predicate, or object. Thus, it was permissible to
claim that the square root function~$\lfloor \sqrt x \rfloor$ is definable
by defining with a formula using $<$ and~$\le$, since the definable
symbols $<$ and~$\le$ could be replaced with the use of formulas
over the original language ($0, S, +, \cdot$). In particular,
$\lfloor \sqrt x \rfloor$ is defined by
\[
\exists x_3 \, (x_2 \cdot x_2 + x_3 = x_1) \land \exists x_3\, (x_1 + S(x_3) = S(x_2) \cdot S(x_2)) .
\]

It is easy to replace defined predicate symbols with their definition. It is
a little
more difficult to replace defined objects or defined functions with their definitions.
The next section will describe formally how to remove
defined objects and functions; in the meantime,
the next example illustrates the general principle.

\begin{example}
Let $\mathfrakA$ be an $L$-structure. Suppose that, in $\mathfrakA$, an object~$a$ is defined
by the formula $B_a(x_1)$, a unary function~$f$ is defined by
the formula $B_f(x_1,x_2)$, and a binary function~$g$ is defined
by $B_g(x_1,x_2,x_3)$.  Let $P$ be a unary predicate symbol in~$L$. Then
$P(g(f(x),a)$ is can be expressed in~$\mathfrakA$ by the formula
\[
\exists u\, \exists v\, \exists w\, (B_a(u) \land B_f(x,v) \land  B_g(v,u,w) \land P(w)).
\]
Note that $B_a(u)$ enforces that $u=a$, and $B_f(x,v)$ enforces that $f(x)=v$,
and finally $B_g(v,u,w)$ enforces that $w = g(v,u) = g(f(x),a)$.
\end{example}
