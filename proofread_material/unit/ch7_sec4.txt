¥section{The First Incompleteness Theorem}¥label{sec:FirstIncompleteness}
¥index{First Incompleteness Theorem|(}

We now give the statement and
proof of G{¥"o}del's First Incompleteness Theorem.
We will prove several forms of the First Incompleteness
Theorem for theories‾$T$ that extend‾$¥TR$. This includes theories‾$T$
such as Robinson's theory‾$¥TQ$ and Peano arithmetic‾$¥PA$, since
we shall later show (in Theorem‾¥ref{thm:QimpliesR}) that
$¥TQ ¥vDash ¥TR$.

The proof of the First Incompleteness Theorem will depend
on the following theorem, which will be proved
later in Section‾¥ref{sec:representTM}:

¥begin{theorem}¥label{thm:decidableAreRepresentable}
Every Turing decidable relation on‾$¥mathbb N$ is representable
in the theory‾$¥TR$. Every Turing computable function on‾$¥mathbb N$
is representable in‾$¥TR$.
¥end{theorem}

By Theorem‾¥ref{thm:TextendsRrepresents}, the same holds
for every axiomatizable theory‾$T$ extending‾$¥TR$. In particular, it holds
for the theory‾$¥TQ$.  Accordingly, the rest of this section
will work with an arbitrary axiomatizable, consistent theory $T¥supseteq R$.

We will first discuss how to encode an $L_¥PA$-term‾$t$ or an $L_¥PA$-formula‾$A$ with
an integer $¥lgn t ¥rgn$ or‾$¥lgn A ¥rgn$, which is
called the ``G{¥"o}del number'' of $t$ or‾$A$.
We will also discuss the representability of the substitution function
$¥TSub(A, x, t)$.  We then state and prove the Incompleteness
Theorem. In fact, we will give three proofs. The first proof
works directly off the undecidability of the halting problem; it
proves the First Incompleteness Theorem for true theories and more
generally for ``$¥omega$-consistent'' theories.
The second and third proofs establish a stronger form
of the First Incompleteness Theorem. These two proofs are variations of
each other; they both use Cantor-style diagonalization.
The third proof gives the full construction
of ``self-referential'' formulas that can be obtained via
a new version of the Diagonal Theorem that applies to
first-order formulas.

¥subsection{G{¥"o}del numbering, numerals, and substitution.}¥index{Godel number@{G{¥"o}del number}}

Theories of arithmetic such as $¥TR$, $¥TQ$ and‾$¥PA$
deal only with integers; however, it is possible for
them to deal indirectly with syntactic metamathematical objects
such as terms, formulas, and proofs by
using G{¥"o}del numbers to represent metamathematical
objects.  Section‾¥ref{sec:AlgorithmsFO} already discussed
representing metamathematical objects with strings of symbols over
a finite alphabet.  We now want to represent metamathematical
object with integers: this will be done similarly to Section‾¥ref{sec:AlgorithmsFO},
but differs by using strings over the alphabet‾$¥{0,1¥}$ which are interpreted as
binary representations of integers.

The details of how G{¥"o}del numbers represent
terms, formulas, and proofs are not terribly important, as long
as it is sufficiently constructive so that algorithms (Turing machines)
can recognize, parse, and manipulate them. For concreteness,
we propose one possible way to define G{¥"o}del numbers for the
first-order language‾$L_¥PA$.
Terms, formulas,
and $¥FO$-proofs in the language‾$L_¥PA$
can be viewed straightforwardly as strings of characters
over the 14‾symbol alphabet
¥[
¥Delta ‾=‾ ¥{ ¥,¥lnot,¥, ¥limplies,¥, ¥forall,¥, (,¥, ),¥,
               x,¥GNzero,¥, ¥GNone,¥, 0,¥, S,¥, +,¥, ¥cdot,¥,
               =,¥, ¥mathrm{comma} ¥}
¥]
There are two different zeroes in the list: the symbol‾$¥GNzero$‾is used when
writing indices of variables in binary notation, whereas $0$‾is
used as the non-logical symbol whose standard interpretation is
the integer‾$0$.¥footnote{So we are using four different concepts
of zero: ``$¥GNzero$''‾is the alphabet symbol used for zeros in subscripts, 
and ``0''‾is is used variously
as a non-logic symbol, as an alphabet symbol, or as
the actual integer zero.}
Variables $x_i$ are represented by a string‾$x$ followed by the binary representation
of‾$i$; e.g., the terms $x_0$, $x_3$ and‾$x_5$ are represented by the strings
$x¥GNzero$, $x¥GNone¥GNone$ and $x¥GNone ¥GNzero ¥GNone$.
This convention for coding variables and their subscripts allows
terms, formulas, sentences, and proofs over
the language of‾$¥PA$ to be straightforwardly written as strings over‾$¥Delta^*$.

To further convert strings from‾$¥Delta^*$ to integers, we use a fixed
length 4-bit binary encoding of symbols in‾$¥Delta$ to encode
a string in‾$¥Delta^*$ as a base‾2 integer.
There are 14 symbols in‾$¥Delta$.
Each symbol of‾$¥Delta$ can be represented by a distinct
4-bit fixed length code word; e.g. ``$¥lnot$'' is encoded by ``0001'', and
``$¥limplies$'' by ``0010'', ``$¥forall$'' by
 ``0011'', etc., up through
``1110'' encoding a comma. (The commas are used to separate formulas
in G{¥"o}del numbers of proofs.)
For example, the formula $x_2 = 0$ would be encoded
as in string‾$¥Delta^*$ as ``$x ¥GNone ¥GNzero {=} 0$'', and then further
encoded
by the binary string ``0110¥kern2pt 1000¥kern2pt 0111¥kern2pt 1101¥kern2pt 1001''.
This is the base‾2 representation of the base‾10 number‾427993.
Thus, the G{¥"o}del number‾$¥lgn x_2=0 ¥rgn$
of ``$x_2=0$'' is the integer 428009.

G{¥"o}del numbers of terms, formulas, proofs, etc.¥ can certainly be
handled by algorithms (Turing machines). For instance, the set
of integers that are valid G{¥"o}del numbers of $¥PA$ terms is
decidable. This is simply because there are algorithms
for recognizing and parsing terms. Similarly, the sets of G{¥"o}del numbers of
formulas and of sentences are decidable.

Theorem‾¥ref{thm:SyntaxDecidable} already discussed a variety
of decidable relations and computable functions
acting on G{¥"o}del numbers of formulas.
We will need several additional computable functions that act on G{¥"o}del numbers
of terms and formulas.  The first one is the 3-ary substitution
function‾$¥TSub$ which accepts as input the G{¥"o}del numbers
$¥lgn A ¥rgn$, $¥lgn x_i¥rgn$ and‾$¥lgn t ¥rgn$ of a formula‾$A$,
a variable‾$x_i$ and a term‾$t$ and produces the G{¥"o}del
number $¥lgn A(t/x_i) ¥rgn$:
¥[
¥TSub(¥lgn A ¥rgn, ¥lgn x_i¥rgn, ¥lgn t ¥rgn) ‾:=‾ ¥lgn A(t/x_i) ¥rgn.
¥]
Another important function is the $¥TNum$ function that
computes the G{¥"o}del numbers of numerals¥index{numeral}.  Namely,
¥[
¥TNum( n ) ‾:=‾ ¥lgn ¥underline n ¥rgn .
¥]
For instance, $¥TNum(3) = ¥lgn S(S(S(0))) ¥rgn$.

The set of G{¥"o}del numbers of valid proofs of an axiomatizable
theory is also
decidable.  More precisely, let $T$ be an axiomatizable theory
in the language of‾$¥PA$. We define $¥TProof_T$, the set of
(G{¥"o}del numbers of) $T$-proofs,
to be the unary
relation defined by
¥[
¥TProof_T(n) ‾¥Leftrightarrow‾ ¥hbox{$n$ is the G{¥"o}del number of a valid $T$-proof}.
¥]
This is decidable since there is an algorithm for recognizing
syntactically correct $T$-proofs.
We further define $¥TPrf_T$ to be the binary relation which
holds for exactly the pairs $¥langle ¥lgn P ¥rgn, ¥lgn A ¥rgn ¥rangle$
such that $P$‾is a valid $T$-proof of the formula‾$A$;
namely,
¥[
¥TPrf_T(n,m) ‾¥Leftrightarrow‾ ¥hbox{¥parbox[t]{4in}{$n$ is valid $T$-proof‾$P$ and the final
formula in‾$P$ has G{¥"o}del number‾$m$.}}
¥]
$¥TPrf$ is also decidable since there is an algorithm which
can recognize valid $T$-proofs and extract their final formula.

Finally, we define the set $¥TThm_T$ of theorems of‾$T$. This set however
is not in general decidable, only computably enumerable.  Specifically,
¥[
¥TThm_T(m) ‾¥Leftrightarrow‾¥hbox{$m =¥lgn A¥rgn$ for a formula $A$ such that $T¥vdash A$}.
¥]
$¥TThm_T$ is computably enumerable since it can be semidecided
by an algorithm that does a brute force search for a $T$-proof of the
formula‾$A$ with G{¥"o}del number‾$n$. An alternate way to define
$¥TThm_T$ is:
¥[
¥TThm_T(m) ‾¥Leftrightarrow‾ ¥exists y¥, ¥TPrf_T(y, m) .
¥]
This definition of $¥TThm_T$ as an existential quantification of $¥TPrf_T$
shows that $¥TThm_T$ is computably enumerable (by
Exercise‾¥ref{exer:projectionCE}).

¥subsection{Undecidability of true theories of arithmetic}

We are now ready to give the first proof of the First Incompleteness
Theorem. This proof will apply to ``true'' theories of arithmetic,
and more generally to ``$¥omega$-¥penalty10000 consistent'' theories. It is
based on a direct many-one reduction to‾$¥THalt_0^¥TTM$, the halting problem
for Turing machines.

¥begin{definition}
A theory‾$T$ is a ¥emph{¥idxnm{true theory} of arithmetic} if it
has language‾$L_¥PA$ and contains only sentences that are
true about‾$¥mathcal N$. The latter condition is equivalent to
$T ¥subseteq ¥TTh{¥mathcal N}$.
¥end{definition}

¥begin{definition}
Let $T$ be a theory with language $L_¥PA$. The theory‾$T$
is ¥emph{$¥omega$-¥penalty10000 inconsistent}¥index{omega inconsistent@{$¥omega$-inconsistent}}
provided the following holds for some formula‾$A(x_i)$
with one free variable‾$x_i$:
¥begin{enumerateLoose}
¥item $T$‾proves $¥exists x_i¥, A(x_i)$, and
¥item For all $n ¥in ¥mathbb N$, $T$ proves $¥lnot A(¥underline n)$.
¥end{enumerateLoose}
We say that $T$ is
¥emph{$¥omega$-consistent}¥index{omega consistent@{$¥omega$-consistent}}
provided that $T$ is not $¥omega$-inconsistent.
¥end{definition}

It is clear that $¥TTh{¥mathcal N}$ is $¥omega$-consistent. Hence
any true theory of arithmetic is $¥omega$-consistent.  This is because
it cannot be the case that every $A(¥underline n)$ is false in‾$¥mathcal N$
while $¥exists x_i ¥, A(x_i)$ is true. Also, note that a
$¥omega$-consistent theory must be consistent.

The next theorem states our first version of the
First Incompleteness Theorem. It is stated for
$¥omega$-consistent theories, but of course applies
to any true theory that extends‾$¥TR$.

¥begin{theorem}[First Incompleteness Theorem --- version 1]
¥label{thm:FirstIncompletefirst}
Suppose $T ¥supseteq ¥TR$ is an axiomatizable, $¥omega$-consistent theory.
Then $T$ is not decidable.
¥end{theorem}

¥begin{proof}
Since $T$ extends‾$¥TR$,
Theorems ¥ref{thm:TextendsRrepresents} and‾¥ref{thm:decidableAreRepresentable}
tell us that every decidable relation is representable in‾$T$.
Section‾¥ref{sec:universalTM} earlier
defined the G{¥"o}del number‾$¥lgn M ¥rgn$ of a Turing machine‾$M$ to be
a binary string. The initial symbol of‾$¥lgn M ¥rgn$ is‾$1$, so we can
view $¥lgn M ¥rgn$ also as being the integer whose base‾2 representation
is given by‾$¥lgn M ¥rgn$. We write $¥underline{¥lgn M ¥rgn}$ to
be the numeral for the integer‾$¥lgn M ¥rgn$; namely, if $i=¥lgn M¥rgn¥in ¥mathbb N$, then
$¥underline{¥lgn M ¥rgn}$ denotes the numeral‾$¥underline i$.

Let $¥THS$ be the decidable binary relation such that, for $¥lgn M ¥rgn$
the G{¥"o}del number of a Turing machine‾$M$,
¥[
¥THS (¥lgn M ¥rgn, n) ‾¥Leftrightarrow‾ ¥hbox{$M(¥epsilon)$ halts in $¥le n$ steps}.
¥]
Here ``$¥THS$'' stands for ``Halts-Steps''. $¥THS$‾is a decidable
relation, since it can be decided by using a universal
Turing machine‾$U$ and running $U(¥lgn M ¥rgn, ¥epsilon)$ until it has
simulated $M(¥epsilon)$ for $n$‾steps. This will discover whether $M$‾has halted
within at most $n$‾steps. Since $¥THS$ is decidable, it is representable
in‾$T$, by a formula $A_¥THS(x,y)$.

The predicate $¥THalt^¥TTM_0$, expressing the halting problem
for Turing machines, is undecidable. We claim that $¥THalt^¥TTM_0 (¥lgn M ¥rgn)$
is true if and only if $T ¥vdash ¥exists y¥, A_¥THS(¥underline{¥lgn M ¥rgn}, y)$.
To prove this,
first suppose that $¥THalt^¥TTM_0 (¥lgn M ¥rgn)$ holds and $M(¥epsilon)$ halts.
Then $M(¥epsilon)$ halts within some number‾$n$ of steps. Since
$A_¥THS$ represents‾$¥THS$, this means
$T¥vdash A_¥THS(¥underline{¥lgn M ¥rgn}, ¥underline n)$
and thus of course $T¥vdash ¥exists y¥, A_¥THS(¥underline{¥lgn M ¥rgn},y)$.

To prove the converse direction, suppose that
$T¥vdash ¥exists y¥, A_¥THS(¥underline{¥lgn M ¥rgn},y)$ but
that $¥THalt^¥TTM_0 (¥lgn M ¥rgn)$ does not hold.
The latter condition implies that $¥THS(¥lgn M¥rgn, n)$ is
false for all‾$n$.  Since $A_¥THS$ represents‾$¥THS$, that implies that
$T ¥vdash ¥lnot A_¥THS(¥underline{¥lgn M¥rgn}, ¥underline n)$.
This contradicts the $¥omega$-consistency of‾$T$ and finishes the proof
of the claim.

We have shown that $¥THalt^¥TTM_0 (¥lgn M ¥rgn)$
is true if and only if $T ¥vdash ¥exists y¥, A_¥THS(¥underline{¥lgn M ¥rgn}, y)$.
In other words, $¥THalt^¥TTM_0 (¥lgn M ¥rgn)$
is true if and only if $¥lgn A_¥THS(¥underline{¥lgn M ¥rgn}, y)¥rgn ¥in ¥TThm_T$.
Since the function mapping $¥lgn M ¥rgn$ to the G{¥"o}del
number of the formula $A_¥THS(¥underline{¥lgn M ¥rgn}, y)$ is computable, and
since $¥THalt^¥TTM_0 $ is undecidable, it must be that $¥TThm_T$ is
undecidable.  That proves Theorem‾¥ref{thm:FirstIncompletefirst}.
¥end{proof}

¥subsection{Undecidability via diagonalization}

We now prove a stronger version of the First Incompleteness Theorem
that applies to theories that may not be true or even $¥omega$-consistent.

¥begin{theorem}[First Incompleteness Theorem --- version 2]
¥label{thm:FirstIncompletesecond}
Suppose $T ¥supseteq ¥TR$ is an axiomatizable, consistent theory.
Then $T$ is not decidable.
¥end{theorem}

¥begin{corollary}
There is no consistent and decidable theory‾$T$ extending‾$¥TR$.
¥end{corollary}

¥begin{proof}[Proof of the corollary]
Any decidable theory is axiomatizable since the theory‾$T$ itself
can be taken as the set of axioms. Thus the corollary follows immediately 
from Theorem‾¥ref{thm:FirstIncompletesecond}.
¥end{proof}

¥begin{proof}[Proof of Theorem‾¥ref{thm:FirstIncompletesecond}]]
Suppose for the sake of a contradiction that $T$‾is decidable,
that is, the set‾$¥TThm_T$ is decidable.
Define the unary function
$¥TSelfSub$ by
¥[
¥TSelfSub(m) ‾:=‾ ¥TSub(m, ¥lgn x_1¥rgn, ¥TNum(m) ) .
¥]
To understand this, 
recall that $¥TNum(m)$ is equal to‾$¥lgn ¥underline m ¥rgn$, the
G{¥"o}del number of the numeral‾$¥underline m$.
We are interested in the case where $m$‾is the G{¥"o}del number‾$¥lgn C ¥rgn$
of a formula‾$C=C(x_1)$
with‾$x_1$ the only variable appearing free in‾$C$.
In this case, $¥TSelfSub(¥lgn C ¥rgn)$ is equal
the G{¥"o}del number of the formula $C(¥underline {¥lgn C ¥rgn}/x_1)$;
i.e.,
¥[
¥TSelfSub(¥lgn C ¥rgn)
    ‾=‾ ¥TSub( ¥lgn C ¥rgn , ¥lgn x_1¥rgn, ¥TNum(¥lgn C ¥rgn ) )
    ‾=‾ ¥lgn C(¥underline {¥lgn C ¥rgn}) ¥rgn .
¥]
Define $¥TThmSS$ to be the unary relation on‾$N$ defined by
¥[
¥TThmSS_T(m) ‾¥Leftrightarrow‾ ¥TThm_T(¥TSelfSub(m)) .
¥]
Since $¥TSelfSub$ is a computable function and since we
have assumed that $¥TThm_T$ is decidable, the relation
$¥TThmSS_T$ is also decidable. Hence $¥TThmSS_T$ is representable
in‾$¥TR$, and hence representable in‾$T$, by some formula $A_{¥TThmSS_T}(x_1)$.
Let $B_{¥TThmSS_T}(x_1)$ be the formula $¥lnot A_{¥TThmSS_T}(x_1)$.

We obtain a contradiction as follows:
¥begin{implications}
¥multicolumn{3}{l}{$T¥vdash B_{¥TThmSS_T}(¥underline{¥lgn B_{¥TThmSS_T}¥rgn})$} & ¥¥
& $¥Leftrightarrow$
    & $T¥vdash ¥lnot A_{¥TThmSS_T}(¥underline{¥lgn B_{¥TThmSS_T}¥rgn})$
    & Definition of $B_{¥TThmSS_T}$ ¥¥
& $¥Leftrightarrow$
    & $¥lgn B_{¥TThmSS_T}¥rgn ¥notin ¥TThmSS_T$
    & $A_{¥TThmSS_T}$ represents $¥TThmSS_T$ ¥¥
& $¥Leftrightarrow$
    & $T¥nvdash B_{¥TThmSS_T}(¥underline{¥lgn B_{¥TThmSS_T}¥rgn})$
    & Definition of $¥TThmSS_T$
¥end{implications}
That completes the proof of Theorem‾¥ref{thm:FirstIncompletesecond}.
¥end{proof}

The intuition behind this proof is that the formula
$B_{¥TThmSS_T}(¥underline{¥lgn B_{¥TThmSS_T}¥rgn})$
is indirectly referring to itself and saying that it is not
a theorem of‾$T$. This is reminiscent of Cantor's
¥idxnm{diagonal argument} for the uncountability of the reals.
The next section gives another proof, which uses a
self-referential formula‾$D_A$ based on a
strengthened form of the ¥idxnm{diagonal argument}.

¥subsection{Undecidability via self-reference}

We now give a second proof of the second version
of the First Incompleteness Theorem,
Theorem‾¥ref{thm:FirstIncompletesecond}. This proof is based on
a self-referential formula. This proof can be viewed
as analogous to the second proof of the undecidability
of the halting problem in Section‾¥ref{sec:SelfReferAlg}. Indeed,
it uses a Diagonal Theorem that is very similar in spirit
to the Diagonal Theorem‾¥ref{thm:Diag} for Turing machines.

¥begin{theorem}[Diagonal Theorem for Theories of Arithmetic]¥label{thm:DiagTR}
¥ ‾ ¥newline
Let $A(x_1)$ be an $L_¥PA$-formula.  There is
a formula $D_A$ such that $¥TR$ proves
¥[
D_A ¥liff A(¥underline{¥lgn D_A ¥rgn}) .
¥]
¥end{theorem}

Thus, provably in‾$¥TR$, the formula $D_A$‾expresses the condition
that the property defined by‾$A(x)$ holds for $x$ equal to
the G{¥"o}del number of‾$D_A$ itself.
It is somewhat remarkable that this equivalence of
$D_A$ and‾$A(¥underline{¥lgn D_A ¥rgn})$ is provable in the very
weak theory‾$¥TR$. This illustrates, however, how
fundamental the Diagonal Lemma is.

¥begin{proof}
Recall the function $¥TSelfSub$
from the previous proof.  We would like to define $E_A(x_1)$ to be the
formula $A(¥TSelfSub(x_1))$; however, it is not possible
to do this directly since $¥TSelfSub$ is an integer function,
not a function symbol of first-order logic.  Instead,
we let the formula $A_{¥TSelfSub}(x_1,y)$ represent the function $¥TSelfSub$ in‾$¥TR$, and
define $E_A(x_1)$ by
¥begin{equation}
¥label{eq:defnEA}
E_A(x_1) ‾:=‾ ¥exists y¥, [ A_¥TSelfSub(x_1,y) ¥land A(y) ].
¥end{equation}
Note how $E_A(x_1)$ states indirectly that $A(¥TSelfSub(x_1))$ holds.

Let $D_A$ be the formula $E_A(¥underline{¥lgn E_A ¥rgn})$, so that
the G{¥"o}del number‾$¥lgn D_A¥rgn$ of‾$D_A$ is equal to $¥TSelfSub(¥lgn E_A¥rgn)$,
and $D_A$‾is the formula
¥[
¥exists y¥, [ A_¥TSelfSub(¥underline{¥lgn E_A ¥rgn},y) ¥land A(y) ] .
¥]
Since $A_{¥TSelfSub}(x_1,y)$ represents $¥TSelfSub$,
¥[
¥TR ‾¥vdash‾ A_¥TSelfSub(¥underline{¥lgn E_A¥rgn}, y) ¥liff y = ¥underline {¥lgn D_A ¥rgn} .
¥]
Therefore, by Theorem‾¥ref{thm:substitution}(b) on substitution,
$¥TR$ proves $D_A ¥liff A(¥underline{¥lgn D_A ¥rgn})$.
That finishes the proof of the Diagonal Theorem.
¥end{proof}

The second proof of Theorem‾¥ref{thm:FirstIncompletesecond}
is in essence the same as the first proof, but it uses
the Diagonal Theorem explicitly:

¥begin{proof}[Second proof of Theorem‾¥ref{thm:FirstIncompletesecond}]
Suppose for the sake of a contradiction that $T$‾is decidable,
that is, the set‾$¥TThm_T$ is decidable. Hence the
complement $¥overline{¥TThm_T}$ is also decidable.
Let
the formula‾$A(x_1)$ represent‾$¥overline{¥TThm_T}$ in the theory‾$¥TR$
and hence in the theory‾$T$. The intuition is that $A(x_1)$ says
¥[
¥hbox{``$x_1$‾is not provable in‾$T$'',}
¥]
or more accurately, 
¥[
¥hbox{``$x_1$‾is not the G{¥"o}del number of a theorem of‾$T$''.}
¥]

Let $D_A$ be the formula from the Diagonal Theorem‾¥ref{thm:DiagTR}
such that $¥TR$‾proves $D_A ¥liff A(¥underline{¥lgn D_A ¥rgn})$.
Then
¥begin{align*}
T ¥vdash D_A &‾‾¥Leftrightarrow‾‾ ¥lgn D_A ¥rgn ¥in ¥TThm_T
      &&¥hbox{Definition of $¥TThm_T$} ¥¥
&‾‾¥Leftrightarrow‾‾ T ¥vdash ¥lnot A(¥underline{¥lgn D_A¥rgn})
      &&¥hbox{$A$ represents $¥overline{¥TThm_T}$} ¥¥
&‾‾¥Leftrightarrow‾‾ T ¥vdash ¥lnot D_A
      &&¥hbox{$¥TR ¥vdash D_A¥liff A(¥underline{¥lgn D_A ¥rgn})$ and $T¥supseteq ¥TR$}. ¥¥
¥end{align*}
This contradicts the consistency of‾$T$ and thereby completes the second proof
of Theorem‾¥ref{thm:FirstIncompletesecond}.
¥end{proof}

Note how the formula‾$D_A$ was chosen to express the condition
that $D_A$‾is not a theorem of‾$T$. Loosely speaking, $D_A$‾says
of itself, ``I am not $T$-provable''. The above proof uses this fact, plus
the hypothesis that the property of being ``$T$-provable'' is decidable,
to obtain a contradiction.

¥begin{corollary}
¥ ‾
¥begin{itemize}
¥item[(a)] $¥TR$, $¥TQ$ and $¥PA$ are not complete and not decidable.
¥item[(b)] $¥TTh {¥mathcal N}$ is not axiomatizable and not decidable.
¥end{itemize}
¥end{corollary}
¥begin{proof}
Part‾(a) follows from the fact that $¥TR$, $¥TQ$ and‾$¥PA$
are axiomatizable and consistent and by Theorems
¥ref{thm:completeDecidable2} and‾¥ref{thm:FirstIncompletesecond}.

Part‾(b) follows by the fact that $¥TTh {¥mathcal N}$ is complete
and consistent, and again by Theorems
¥ref{thm:completeDecidable2} and‾¥ref{thm:FirstIncompletesecond}.

Alternatively, since all four theories $¥omega$-consistent,
the corollary can be proved using
Theorem‾¥ref{thm:FirstIncompletefirst} instead of
Theorem‾¥ref{thm:FirstIncompletesecond}
¥end{proof}

¥begin{corollary}¥label{coro:notAllTrue}
Let $T$ be a consistent, axiomatizable theory extending‾$¥TR$.
There is a true sentence‾$C$ (that is, $¥mathcal N ¥vDash C$) such
that $T ¥nvdash C$.
¥end{corollary}

Thus, for instance, $¥PA$‾does not prove all sentences that are
true about the integers. We will see some explicit examples of
such formulas‾$C$ when discussing the Second Incompleteness
Theorem in Section‾¥ref{sec:SecondIncomplete}.

¥begin{proof}
By the First Incompleteness Theorem, $T$‾is not complete. Hence there
is some sentence‾$B$ such that neither $B$ nor‾$¥lnot B$ is a consequence
of‾$T$. One of $B$ and‾$¥lnot B$ is true and is the desired formula‾$C$.
¥end{proof}

¥begin{definition}
Work with an arbitrary first-order language‾$L$.
A sentence‾$B$ is ¥idxem{independent} of‾$T$ if
neither $B$ nor‾$¥lnot B$ is a consequences of‾$T$.
¥end{definition}

Note that $T$ is not complete if and only if there
is an independent sentence‾$B$.
¥smallskip

The previous proof established:

¥begin{corollary}
Let $T$ be a consistent, axiomatizable theory extending‾$¥TR$.
Then there is a sentence‾$C$ which is independent of‾$T$.
¥end{corollary}

¥subsection{Undecidability of pure first-order logic}

An immediate consequence of
the undecidability of the finitely axiomatized theory‾$¥TQ$
is that the pure first-order logic in the
language of‾$¥PA$ is undecidable.
This is established by the next theorem.

¥begin{theorem}¥label{thm:UncidablePA}
The set of valid $L_¥PA$-sentences is undecidable.
¥end{theorem}

¥begin{proof}
Let $¥TQ_{1-7}$ be the conjunction of the seven axioms of‾$¥TQ$,
¥[
¥TQ_{1-7} ‾:=‾ ¥TQ_1 ¥land ¥TQ_2 ¥land ¥TQ_3 ¥land ¥TQ_4 ¥land ¥TQ_5 ¥land ¥TQ_6 ¥land ¥TQ_7 .
¥]
A sentence‾$A$ is provable in‾$¥TQ$ if and only if $¥TQ_{1-7}¥limplies A$ is valid.
Therefore, the mapping $¥lgn A ¥rgn ¥mapsto ¥lgn ¥TQ_{1-7}¥limplies A ¥rgn$ is
a many-one reduction from theory‾$¥TQ$ to the set of logically valid $L_¥PA$-sentences.
Since $¥TQ$ is undecidable, the set of logically valid $L_¥PA$-sentences is undecidable.
¥end{proof}

The statement of
Theorem‾¥ref{thm:UncidablePA} does not depend on any properties of the
integers at all.  All it uses is the fact that the
language contains a constant symbol and
two binary function symbols. In fact, the same theorem holds
even for the case where the language‾$L$ consists
of a single binary predicate symbol. This can be proved by
letting the only non-logical symbol be‾$¥in$ and showing that
there is a finitely axiomatized fragment of Zermelo-Fraenkel set
theory that can interpret the theory‾$¥TQ$. Describing
Zermelo-Fraenkel set theory and what it means to interpret‾$¥TQ$
is beyond the scope of the present text however.¥footnote{For
the reader with knowledge of set theory: One way to argue this is that
Zermelo-Fraenkel set theory‾$¥ZF$ can interpret‾$¥PA$.
Hence, a priori, some finite fragment of‾$¥ZF$
can interpret‾$¥TQ$.}

¥subsection{Undefinability of truth}

The above results imply that the theory $¥TTh {¥mathcal N}$ is
not decidable, so there is no algorithm that can decide whether
an arbitrary given sentence is true in the integers.
In fact, there are many sets that can be
defined in‾$¥mathcal N$ that are not decidable.  A prominent
example is the set of logical consequences of‾$¥PA$ itself.
Namely, let $¥scPrf_¥PA(x_1, x_2)$
be a formula that represents the binary predicate‾$¥TPrf_¥PA$ in‾$¥TR$.¥footnote{Earlier
we would have used the notation $A_{¥TPrf_¥PA}$ instead of $¥scPrf_¥PA$.}
Then let $¥scThm_¥PA$ be the formula
¥[
     ¥exists y¥, ¥scPrf_¥PA (¥lgn A ¥rgn, y).
¥]
Clearly, $¥scThm_¥PA$ is an $L_¥PA$-formula that
defines the set of formulas provable from the axioms of‾$¥PA$.
So this set is definable in‾$¥mathcal N$ even though, by the First
Incompleteness Theorem, it is not decidable.

This raises the question of whether the theory $¥TTh {¥mathcal N}$
is definable in‾$¥mathcal N$. That is, the question
of whether there is a formula‾$¥TTr(x_1)$ such that
$¥mathcal N ¥vDash ¥TTr(¥lgn A ¥rgn)$ holds for exactly the
sentences‾$A$ which are true in‾$¥mathcal N$.

Such a formula would be called a 
``definition of truth''¥index{truth, definition of}. 
Tarski's
theorem on the undefinability of truth shows that such a definition of truth
does not exist.

¥begin{theorem}[Undefinability of Truth]¥label{thm:truthUndefinable}
There is no formula $¥TTr(x_1)$ such that for all sentences‾$A$,
$¥mathcal N ¥vDash ¥TTr(¥lgn A ¥rgn)$ holds if and only if $¥mathcal N ¥vDash A$.
¥end{theorem}

In other words, there is no formula‾$¥TTr(x_1)$ such that for all sentences‾$A$,
$¥mathcal N ¥vDash A ¥liff ¥TTr(¥lgn A ¥rgn )$.

¥begin{proof}
The proof is by contradiction. Assume that there is a definition
of truth‾$¥TTr(x_1)$.
% For any sentence‾$A$, either
% $¥mathcal N ¥vDash A$ and $¥mathcal N ¥vDash ¥TTr(¥lgn A ¥rgn)$ hold,
% or $¥mathcal N ¥nvDash A$ and $¥mathcal N ¥vDash ¥TTr(¥lgn ¥lnot A ¥rgn)$ hold.
By the Diagonal Theorem‾¥ref{thm:DiagTR}, there is
a sentence $D_{¥lnot ¥TTr}$ such that
$¥TR$‾proves
¥[
D_{¥lnot ¥TTr} ¥liff ¥lnot ¥TTr( ¥underline{¥lgn D_{¥lnot ¥TTr} ¥rgn} ) .
¥]
Since $¥TR$ is a true theory, this equivalence holds in‾$¥mathcal N$.
We have
¥begin{align*}
¥mathcal N ¥vDash D_{¥lnot ¥TTr}
   & ‾‾¥Leftrightarrow‾‾ ¥mathcal N ¥vDash ¥lnot ¥TTr( ¥underline{¥lgn D_{¥lnot ¥TTr} ¥rgn} )
       &&¥hbox{By choice of‾$D_{¥lnot ¥TTr}$} ¥¥
   &  ‾‾¥Leftrightarrow‾‾ ¥mathcal N ¥nvDash D_{¥lnot ¥TTr}
       && ¥hbox{$¥TTr$ is a definition of truth}
¥end{align*}
This is a contradiction and finishes the proof of Theorem‾¥ref{thm:truthUndefinable}.
¥end{proof}
