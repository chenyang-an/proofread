\section{Semantic Theorems on Constants}\label{sec:constantsSemantic}

A common method of constructing informal proofs is to introduce
a new name for some unnamed object. For instance, if we are trying to
prove a universal statement $\forall x\, A(x)$, we might introduce
a new name ``$c$'' to represent an arbitrary fixed value of~$x$. If we then
succeed in proving that $A(c)$ holds then, since $c$
was arbitrary, we can conclude that $\forall x\, A(x)$ holds.
Dually, if we are given an existential statement $\exists x\, A(x)$ as a hypothesis,
we might introduce a new name ``$c$'' for an arbitrary object representing an $x$ such that
$A(x)$. We then are justified in adding $A(c)$ as a new hypothesis.

These two informal reasoning methods are made formal by the next theorem and
corollary.

\begin{theorem}\label{thm:constantsSemantic}
Let $A(x)$ be a formula and $\Gamma$ be a set of formulas.
Suppose that
the constant symbol~$c$ does not appear in~$A(x)$ or in
any formula in~$\Gamma$. Then
\[
\hbox{$\Gamma\cup \{\exists x\, A(x)\}$ is satisfiable}
\quad \hbox{if and only if}\quad
\hbox{$\Gamma\cup \{A(c)\}$ is satisfiable}.
\]
If, in addition, $B$ is a formula that does not contain~$c$, then
\[
\hbox{$\Gamma,\, \exists x\, A(x) \vDash B$} 
\quad \hbox{if and only if}\quad
\hbox{$\Gamma,\, A(c) \vDash B$} .
\]
\end{theorem}

\begin{corollary}\label{coro:constantsSemantic}
Let $A(x)$ be a formula. Let $\Gamma$ be a set of formulas such that
the constant symbol~$c$ does not appear in $A(x)$ or
any formula in~$\Gamma$. Suppose $\Gamma \vDash A(c)$.
Then $\Gamma \vDash \forall x A(x)$.
\end{corollary}

The corollary is proved by applying the theorem to the
formula $\lnot A$ and using the proof by contradiction
principle of Theorem~\ref{thm:ContradictionSemanticFO}.
The basic idea for the proof of the first part of
Theorem~\ref{thm:constantsSemantic} is rather simple:
if we have a model for $\exists x\, A(x)$, we can enlarge the language
to include $c$ as a new constant symbol, and let the interpretation of~$c$
be some object that makes $A(c)$ true. Such an object is guaranteed to exist
if and only if $\exists x\, A(x)$ is true.  Making this intuition formal takes a little work
and we start
with a rather general definition that will also be useful later.

\begin{definition}
Let $L$ and $L^\prime$ be languages with $L \subseteq L^\prime$. Suppose
$\mathfrakA$~is an $L$-structure and $\mathfrakB$~is an $L^\prime$-structure.
We say that $\mathfrakB$~is an \idxem{expansion} of~$\mathfrakA$, and that
$\mathfrakA$~is a \idxem{restriction} of~$\mathfrakB$ provided that
\begin{itemize}
\item They have the same universe: $|\mathfrakA| = |\mathfrakB|$, and
\item The interpretations of the symbols in~$L$ are the same in $\mathfrakA$ and~$\mathfrakB$.
In other words, $c^\mathfrakA = c^\mathfrakB$ and
$P^\mathfrakA = P^\mathfrakB$ and
$f^\mathfrakA = f^\mathfrakB$ for all symbols $c, P, f \in L$.
\end{itemize}
\end{definition}

\begin{example}
$\mathcal N' = (\mathbb N, 0, S, +, \cdot, <)$ is an
expansion of $\mathcal N = (\mathbb N, 0, S, +, \cdot)$.
\end{example}

As hinted at in the example, any definable predicate, object, or function of a
structure~$\mathfrakA$ can be used to form an extension of~$A$. Much more
general constructions can be used for expansions of course.  A basic property of
expansions is that the truth of an $L$-sentence is the same in $A$ and~$B$.

\begin{theorem}\label{thm:truthInExpansion}
Let $L$, $L^\prime$, $\mathfrakA$ and $\mathfrakB$ be as above with
$\mathfrakB$ an expansion of~$\mathfrakA$. Let $\sigma$ be any
object assignment.
(Note that although $\mathfrakA$ and $\mathfrakB$ have different languages,
they have the
same object assignments by virtue of having the same universe.)
\begin{enumerate}
\item For any formula~$A$, $\mathfrakA \vDash A[\sigma]$ if and only if $\mathfrakB \vDash A[\sigma]$.
\item For any sentence~$A$, $\mathfrakA \vDash A$ if and only if $\mathfrakB \vDash A$.
\end{enumerate}
\end{theorem}

A formal proof of Theorem~\ref{thm:truthInExpansion} would use induction on the
complexity of the formula~$A$. However, we omit the proof,
since it should be completely clear that the
theorem must hold: Examining the definition of truth, the truth of a formula
in a structure depends only on the universe and on the interpretations
of the non-logical symbols that appear in the formula.
And, the formula~$A$ uses only non-logical
symbols from~$L$, which have the same interpretations in $\mathfrakA$ as in~$\mathfrakB$.
\hfill $\qed$

\begin{proof}[Proof of Theorem~\ref{thm:constantsSemantic}]
Let $L$ be the language containing the non-logical symbols that
appear in $A$ and~$\Gamma$. Let $L^\prime$ be
$L \cup \{c \}$ where $c \notin L$ is a constant symbol.
The first part of the theorem states that
$\Gamma\cup \{\exists x\, A(x)\}$ is satisfied by some $L$-structure~$\mathfrakA$
if and only if
$\Gamma\cup \{A(c)\}$ is satisfied by some $L^\prime$-structure~$\mathfrakB$.

First suppose that an $L^\prime$-structure~$\mathfrakB$ with object assignment~$\sigma$
satisfies $\Gamma\cup\penalty10000 \{A(c)\}$, namely $\mathfrakB \vDash A(c) [\sigma]$.
Then certainly $\mathfrakB \vDash \exists x\, A(x) [\sigma]$. Letting $\mathfrakA$ be the
restriction of~$\mathfrakB$ to the language~$L$, Theorem~\ref{thm:truthInExpansion}
implies that $\mathfrakA \vDash (\Gamma\cup\penalty10000 \{\exists x\, A(x)\})[\sigma]$.

Second, suppose that $(\mathfrakA, \sigma)$ satisfies
$\Gamma\cup \{\exists x\, A(x)\}$. By the definition of truth, there is a
$a\in |\mathfrakA|$ such that $\mathfrakA \vDash A(x)[\tau]$ where
$\tau$ is the $x$-variant of~$\sigma$ with $\tau(x)=a$.  Let
$\mathfrakB$ be the expansion of~$\mathfrakA$ to the language~$L^\prime$
obtained by letting $c^\mathfrakB = a$.  By Theorem~\ref{thm:substitution}(b),
\[
\mathfrakB \vDash (x=c \limplies (A(x)\liff A(c)) [\tau].
\]
Since $\tau(x) = c^\mathfrakB$ and $\mathfrakB\vDash A(x)[\tau]$, we have
$\mathfrakB\vDash A(c)[\tau]$. Since $\tau$~is an $x$-variant of~$\sigma$,
$\mathfrakB\vDash A(c)[\sigma]$. Finally, $\mathfrakB\vDash \Gamma[\sigma]$ by
Theorem~\ref{thm:truthInExpansion} since $\mathfrakA\vDash \Gamma[\sigma]$.
That proves the first part of the theorem.

The second part of the theorem follows immediately since 
$\Gamma, \exists x\,A(x) \vDash B$ holds if and only
$\Gamma, \lnot B, \exists x\, A(x)$ is unsatisfiable and
since $\Gamma, A(c) \vDash B$ holds if and only
$\Gamma, \lnot B, A(c)$ is unsatisfiable.
\end{proof}


The next theorem lets us reexpress a logical implication $\Gamma \vDash A$ involving
formulas as a logical implication involving only sentences.
The general idea is to replace free variables in $\Gamma$ and~$A$
with new constant symbols.
To illustrate this,
consider the fact that $P(x,y) \vDash \lnot \lnot P(x,y)$.
We wish to reexpress this using sentences instead of the formulas involving the
free variables $x$ and~$y$. For this, we introduce two new constant symbols,
say $c$ and~$d$. Then $P(x,y) \vDash\penalty10000 \lnot\lnot P(x,y)$ is equivalent to
$P(c,d) \vDash \lnot \lnot P(c,d)$.

To formalize this generally, we start with an $L$-formula~$A$ and
a set~$\Gamma$ of $L$-formulas. We let $L^\prime$ be $L \cup \{d_1,d_2,d_3,\ldots\}$
with infinitely many new constant symbols. We write $A(\vec d/\vec x)$ and $\Gamma(\vec d/\vec x)$
to denote the result of replacing every free occurrence of every~$x_i$ with~$d_i$.
Thus, $A(\vec d/\vec x)$~is an $L^\prime$-sentence and $\Gamma(\vec d/\vec x)$ is a set
of $L^\prime$-sentences.\footnote{The notation here assumes that the language~$L$ is countable,
and hence the number of formulas in~$\Gamma$ and, in particular, the number of free variables
in~$\Gamma$ is countable. The construction works, however, also for uncountable
languages where $\Gamma$ might use uncountably many variables.}

\begin{theorem}\label{thm:variablesToConstants}
Let $L$, $L^\prime$ and $d_1, d_2, d_3, \ldots$ be as above.
Let $\Gamma$ be a set of $L$-formulas, and $A$~be an $L$-formula. (So the
constant symbols~$d_i$ do not appear in $\Gamma$ or~$A$.)
\begin{itemizeLoose}
\item[(a)] $\Gamma$ is satisfiable if and only if $\Gamma(\vec d/\vec x)$ is satisfiable.
\item[(b)] $\Gamma\vDash A$ if and only if $\Gamma(\vec d/\vec x) \vDash A(\vec d/\vec x)$.
\end{itemizeLoose}
\end{theorem}

\begin{proof}
To state (a) more precisely, it states that there is
an $L$-structure~$\mathfrakA$ and an object assignment~$\sigma$
so that $(\mathfrakA, \sigma)$
satisfies~$\Gamma$ if and only if there is a $L^\prime$-structure~$\mathfrakB$
which satisfies $\Gamma(\vec d / \vec x)$.  The relationship
between $\mathfrakA$ and $\mathfrakB$ is that
$\mathfrakB$~is the expansion of~$\mathfrakA$ to the language~$L^\prime$
with $d_i^\mathfrakB = \sigma(x_i)$.  Reasoning as in the proof of
Theorem~\ref{thm:constantsSemantic} proves the desired equivalence.

Part~(b) is an immediate consequence of part~(a).
\end{proof}
